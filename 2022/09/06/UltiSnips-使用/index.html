使用 `` 包裹起来的部分可以直接写脚本，如 Shell Script, Vimsript, Perl, Python. 语法如下:

  • Shell : `echo hi` or `#! /usr/bin/perl
    print “Hello”`
  • Vimscript : `!v “1 + 2” . string(1+2)`, 也就是前面有一个 !v
  • Python : `!p snip.rc = “Hello”`, 前面有个 !p
