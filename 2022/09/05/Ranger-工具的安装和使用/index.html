Ranger 工具的安装和使用

Github 相关

在 github 上和 archwiki 上有很多有用的资料.

ranger 是一个命令行下的文件管理器 (file manager).

可以在 man ranger 和 HACKING.md 中查看信息.

对于配置，可以查看 ranger/config/ 或者复制默认配置到 ~/.config/ranger with ranger --copy-config.

examples/ 中包含了脚本和插件，这些文件可以在 /usr/share/doc/ranger 中查看。

资料

可以查看 ~/.config/ranger/rc.conf 文件来看有哪些快捷键.

修改这个配置文件可以定义很多快捷键.

部分快捷键

重命名: cw

修改文件名: a 和 A

选中全部, 按 v.

bulkrename 命令, 在 vim 下修改文件名.

显示隐藏文件: zh

显示文件大小: dU

排序: o 加一个键

决定打开方式: r

移动上层文件夹: []

在历史记录中跳转: H L

利用 ranger 的 fzf 插件

复制文件路径: yp
