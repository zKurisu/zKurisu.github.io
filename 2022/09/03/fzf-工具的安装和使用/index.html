fzf 工具的安装和使用

github 相关内容
fzf 是 fuzzy finder 的简称。

用于在命令行下查找文件, 目录等。

注意在下载后： Key bindings (CREL-T/CREL-R/CREL-C) and fuzzy auto-completion may not be enabled by default.

Usage

使用几个快捷键：

  • CTRL-K 和 CTRL-J 用于上下移动
  • Enter 用于选中，CTRL-C 用于退出
  • -m multi-select mode, 使用 TAB 和 Shift-TAB 来标记多个 items
  • 鼠标滚轮也能移动
    被选中的会被输出到 STDOUT 中。

layout

可改变窗口大小，具体见 github.

Search syntax

可以使用空格来进行层次搜索, 比如 Documents/books/hello.txt, 你可以在 fzf 中输入 Do boo hello 来查找。

用 | 代表 “或者”, ^ , $ 和 ! 都可以使用.
