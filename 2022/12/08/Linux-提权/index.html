利用 SUID 二进制文件进行提权

SUID 为 Set UID, 就是设置用户权限.

当你具有 SUID 权限二进制文件的可执行权限时，执行时会获得属主的权限.

SUID 只能添加给二进制文件，且只有在运行时才会获得属主权限.

比如, 你给 rm 命令添加了 SUID 权限，假设 test 文件是 root 用户创建的文件，此时, 你是普通用户，具有 rm 的执行权限， 那么:
