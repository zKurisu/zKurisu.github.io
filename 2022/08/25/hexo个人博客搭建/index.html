hexo个人博客搭建

参考b站程序羊

操作步骤

首先要安装nodejs和npm。

node -v
npm -v

查看版本。

使用npm下载并切换镜像源：

npm install -g cnpm --registry=https://registry.npm.taobao.org

下载hexo-cli：

cnpm install -g hexo-cli

在用户目录下创建一个文件夹，初始化这个文件夹：

sudo mkdir Blog
sudo hexo init

启动hexo

sudo hexo s

hexo基本操纵

一般生成完之后先clean, 然后generate, 最后 server. 将内容推送带github上去就是deploy操作。

创建新文章

hexo new “My New Post”

new可简写为n.
创建的blog所在的位置，source/_posts中。/

启动hexo

hexo server

server可简写为s

文件生成

在创建并写完一个文章之后，要生成一下。

hexo generate

generate可简写为g.

在本地4000端口打开

一般用来测试。

这个是本地的，还未部署到远端。

博客部署到Github上

在github上新建一个仓库。

名称为，以我自己的为例：

zKurisu.github.io

安装git插件：

cnpm install --save hexo-deployer-git

配置_config.yml文件添加：

type: git
repo: github上的仓库
branch: master

部署到远端：

sudo hexo deploy

deploy可简写为d.

这时就可以用GitHub来访问了。

自定义主题

github上的仓库

http://github.com/litten/hexo-theme-yilia

克隆到blog目录中：

sudo git clone ...

然后配置到博客中。
修改_config.yml文件中的Themes。
