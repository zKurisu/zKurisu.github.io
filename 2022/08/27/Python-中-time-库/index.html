Python 中 time 库

time 库是 Python 中处理时间的标准库。

功能

  • 获取系统时间，并格式化输出
  • 计时，方便分析程序的性能

time() 函数

用于获取从1970年1月1日00：00：00开始，到现在的总秒数，也被称为计算机内部时间.

localtime() 函数和 gmtime() 函数

都可返回以元组表示的时间对象.

localtime() 返回当地时间。

gmtime() 返回世界统一时间。

两者的默认参数为 time.time() 的返回值. 接受一个浮点数，表示秒。

ctime() 函数

其会读取当前时间并以易读的格式表示，返回字符串, 默认参数为 time.time() 的返回值:

strftime() 函数

以字符串形式格式化输出时间, 接受两个参数，前者为格式，后者为时间对象:

其格式有:

  • %Y
  • %m
  • %d
  • %B, 完整月份名
  • %b, 简化
  • %A, 完整星期名
  • %a, 简化
  • %H
  • %I, 12 小时制
  • %p, 上下午
  • %M
  • %S

perf_counter()

记录 CPU 运行时间，返回值以秒为单位.
