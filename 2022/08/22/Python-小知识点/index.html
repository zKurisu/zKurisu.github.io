try … except

将可能发生错误的语句放在try模块，用except处理异常。

except 可以指定一个专门的异常，若未指定则默认为所有异常。

每一个try都至少要有一个except.

可以多分支.

若使用 try ... except ... finally, finally 部分无论 try 是否捕获错误都会执行。

except 中的 as 应该是设置别名。
