解决在-shell-脚本中-cd-命令不起作用问题

描述通常我们直接执行 shell 脚本文件, 它会打开一个子 shell 并由其执行, 因此, 我们的 cd 命令实际上是在子 shell 中执行了, 所以当前 shell 中并未跳转. 解决使用 source 命令, 它会读取脚本文件并在当前 shell 里面执行, 并没有创建新的 shell.
