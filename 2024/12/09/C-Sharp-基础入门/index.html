什么是 .NET

.NET 是微软提供的一个开发平台, 为多种编程语言 (C#, VB.NET 等) 提供应用程序开发的库和工具.

.NET 主要由四个部分组成:

  • CLR (Common Language Runtime): 运行时环境, 负责执行 .NET 应用程序, 并提供垃圾回收, 类型安全等服务
  • 类库: 提供丰富的预构建类和函数, 简化开发过程
  • ASP.NET: 用于创建 Web 应用和服务的框架
  • Entity Framework: 用于数据访问的对象关系映射 (ORM) 框架

其实 .NET 平台也分几个:

  • .NET Framework, 由微软开发, 但只能在 Windows 上运行
  • MONO, 由社区开发, 可以跨平台运行
  • .NET CORE, 由微软开发, 可以跨平台运行

C# 和 .NET 的关系

C# 是 .NET 平台上的一种主要编程语言. 开发者使用 C# 编写应用程序, 而这些应用程序在 .NET 环境中运行.

环境准备

.NET 安装

在 Archlinux 上:

  • dotnet-runtime 包含了运行 .NET 应用程序所需的运行时环境
  • dotnet-sdk 应用程序所需的所有工具和库, 包括编译器, 运行时和其他开发工具
  • aspnet-runtime 包含运行 ASP.NET Core 应用程序所需的组件

命令行工具

dotnet

dotnet 命令用于:

  • 创建 C# 项目
  • 编译
  • 运行

创建 C# 项目

  • new 表明创建一个新项目目录
  • console 是模板类型, 表示命令行程序, 还有其他模板, 如:
    • classlib, 类库项目
    • web, ASP.NET Core Web 项目
    • mvc, ASP.NET Core MVC 项目
    • 等
  • -n, --name, 指定项目名

目录结构为:

  • bin/Debug/net8.0/ 放置编译后的可执行文件等
  • obj/ 目录存储中间文件和生成过程中使用的临时文件
  • HelloWorld.csproj 包含项目的配置信息, 目标框架, 依赖项, 编译选项等
  • Program.cs 包含应用程序的入口点 (Main 方法) 和主要逻辑代码
