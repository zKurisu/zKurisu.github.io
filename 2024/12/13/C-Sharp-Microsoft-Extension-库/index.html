基础示例

  • 先用 LoggerFactory 对象创建 ILoggerFactory 接口实例, 并对其配置
  • 再用 ILoggerFactory 实例来创建 ILogger 实例 (其配置由创建自己的 ILoggerFactory 对象决定)
  • 用 ILogger 打印日志信息, 如 LogInformation() 方法
  • ILoggerFactory 是一个用于创建日志记录器实例的接口 (I 就是指 Interface)
  • LoggerFactory.Create(...), 该静态方法用于创建一个 ILoggerFactory 实例, builder 参数传入一个函数来对实例进行配置
  • builder => builder.AddConsole(), 配置日志记录器以将日志输出到控制台
  • ILogger logger = factory.CreateLogger<Program>();, 创建一个 ILogger 实例
    • ILogger 接口用于记录信息, 警告, 错误等
    • factory.CreateLogger<Program>(), 创建一个与 Program 类相关的日志记录器
