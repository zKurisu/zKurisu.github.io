5G-网络架构与组网部署

相关概念

5GC

5GC, 5G Core Network, 指 5G 核心网.

NG-RAN

NG-RAN, Next Generation Radio Access Network, 指 5G 无线接入网.

eNB, ng-eNB, gNB

eNB, evolved NodeB (B 可能是 Base Station), 指 4G 的基站.

ng-eNB, next generation evolved NodeB, 指 4G 增强型的基站, 可以实现部分 5G 基站的功能.

gNB, Next Generation NodeB, 指 5G 基站.

NE

NE, Network Element, 网元. 指在通信网络中具有独立控制和管理能力的节点设备. 它可以是网络中的一个单独设备或模块, 具有特定的功能和任务, 并通过通信协议与其他网元进行交互.

在 5G 网络中, NB, eNB, AMF, UPF 这些都可以被视作网元.

interface between Network Elements 指不同网元之间进行通信和交换信息的接口, 这些接口定义了网元之间的通信协议, 数据格式和消息交互方式, 使得不同网元能够进行数据传输, 控制命令传递和协同操作.

Xn 接口

Xn, eXtended Node interface, 指 5G 之间通信的接口. 一般是接入网中网元间的接口.

NG 接口

NG interface, 指 NG 接口, 一般是接入网与核心网之间的接口.

AMF/UPF/SMF

AMF, UPF, SMF 都是 Core Network 中的核心组件, 分别负责不同的功能.

AMF, Access and Mobility Management Function, 访问与移动性管理功能. 负责处理与用户设备的接入, 认证, 安全性和定位相关的功能, 它是用户与核心网络之间的关键接口, 负责处理用户的注册, 鉴权, 位置跟踪和移动管理等操作.

UPF, User Plane Function, 用户面功能, 负责处理用户数据的转发和处理, 它是5G网络中的数据传输引擎. UPF 负责对用户数据进行路由, 转发, 分组, 解包和封包等操作, 确保用户数据能够在网络中快速, 可靠地传输. UPF 还负责实施网络策略, 如流量控制, 质量保证和安全性等.

SMF, Session Management Function, 会话管理功能, 负责管理用户会话的建立, 维护和释放. 它处理用户的会话请求, 包括创建, 修改和删除会话等操作. SMF 还负责配置策略和服务数据流, 管理会话的安全性和质量, 以及网络切片的支持.

LTE

LTE, Long-Term Evolution, 是一种移动通信技术标准, 也就是 4G (4 Generation), 第四代移动通信系统.

EPC

EPC, Evolved Packet Core network, 演进分组核心网, 是 LTE 网络的 Core Network, 负责处理用户数据的传输和控制, 处理控制平面和用户平面的功能, 提供移动性管理, 会话管理, 策略控制, 数据传输等服务.

E-UTRAN

E-UTRAN, Evolved Universal Terrestrial Radio Access Network, 演进通用陆地无线接入网. 是 LTE 网络中的无线接入部分, 由多个基站 (如eNB) 组成. 它负责与移动设备进行无线通信, 处理数据传输, 接入控制, 移动性管理和无线资源管理等功能.

MME/S-GW

MME, Mobility Management Entity, 是 LTE Core Network 中的网元之一, 负责处理移动性管理和控制平面的功能. 处理设备的鉴权, 用户注册, 位置跟踪, 会话管理等操作.

S-GW, Serving Gateway, 也是 LTE Core Network 中的网元之一, 负责处理用户数据的传输和路由. 它在用户数据与 EPC 之间进行转发, 提供用户数据的路由, 分组, 解包和封包等功能.

S1 和 X2 接口

S1, S1 interface (“S1” 似乎没有具体的扩展含义, 仅作为标识符), 是 LTE 网络中 eNB 与 Core Network 之间的接口. 它承载了控制平面和用户平面的信令和数据传输, 连接了基站和 MME.

X2, X2 interface (“X2” 似乎也没有具体的扩展含义), 是 LTE 网络中不同 eNB 之间的接口, 它允许基站之间进行直接的无线资源协调和通信, 以实现无缝的切换, 干扰协调和传输优化等功能.

BBU, RRU, CU, DU, AAU, 5GC, MEC

BBU, BaseBand Unit, 基带单元. 无线通信系统中的一部分, 主要负责处理基带信号的处理和调度. 它通常位于基站 (如 LTE 基站) 的核心部分, 负责数字信号的处理, 调制解调, 编码解码等功能.

RRU, Remote Radio Unit, 远程无线单元. 无线通信系统中的一部分, 用于无线信号的放大和无线前端处理. 它通常位于基站的远程位置, 与 BBU 通过光纤或同轴电缆进行连接. RRU 负责将数字信号转换为无线信号, 并在无线信号的传输过程中进行放大和处理.

CU, Central Unit, 中央单元. 5G 网络中的一部分, 负责控制面的功能. 它可以理解为 5G 基站中的控制器, 处理设备注册, 鉴权, 移动性管理等操作, 并与 DU 进行通信以控制和管理基站的行为.

DU, Distributed Unit, 分布单元. 5G 网络中的一部分, 负责用户面的功能. 它负责处理用户数据的转发, 路由和传输, 与 CU 进行通信以提供无线接入和数据传输服务. DU 通常与 AAU 或 RRU 集成在一起, 形成一个功能完整的基站.

AAU, Active Antenna Unit, 有源天线单元. 一种集成了天线和无线前端功能的单元. 它具有放大无线信号的能力, 并可以通过数字接口与 BBU 或 DU (下行链路传输单元) 进行连接. AAU 通常用于提高系统的容量和覆盖范围, 并支持多天线技术 (如 MIMO) 以提供更好的无线传输性能.

5GC, 5th Generation Core network, 指第五代移动通信技术的核心网络. 它负责处理移动网络中的用户认证, 会话管理, 移动性管理和服务控制等功能.

MEC, Multi-access Edge Computing, 多接入边缘计算. 是一种计算模型, 将计算和存储资源移动到网络的边缘, 以便更接近终端用户. MEC 允许在网络边缘进行应用程序和服务的部署, 以提供低延迟和更高的带宽效率. 它在 5G 网络中发挥着重要作用, 支持边缘智能, 边缘缓存和低时延应用等场景.

在 5G 的架构中, 5GC 的功能被下放到各 MEC 来处理.
