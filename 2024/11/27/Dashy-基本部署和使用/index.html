配置

主要依靠挂载文件到容器里的 /app/user-data/ 目录下来配置, 该目录下各文件的作用如下 (这里列出一部分):

  • conf.yml, 主配置文件, 这个必须有
  • *.yml, 其他配置文件
  • favicon.ico, 浏览器 Tab 处的图标
  • initialization.html 是 dashy 在初始化时显示的页面
  • *.html, 自定义的 HTML 页面, 可以通过 http://my-dashy/my-page.html 来访问

conf.yml

conf.yml 中有三个 root attributes:

  • pageInfo, Dashboard 的元信息, 如 title, description, nav bar links 和 footer text
  • appConfig, Dashboard settings, 如主题, 认证, 语言, 自定义内容
  • sections, 包含各个要显示的 section
  • pages (??)
