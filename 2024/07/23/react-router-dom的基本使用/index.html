Client Side Routing

Client 指打开页面的用户, 另一侧是 Server, 提供页面.

Client Side Routing 指由用户端浏览器已经加载的 JavaScript 来管理路由, 而不是向服务器请求一个新页面. 这样可以提供更流畅的用户体验, 因为应用程序可以在不重新加载整个页面的情况下更新页面内容.
