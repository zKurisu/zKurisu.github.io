src 目录

这里 index.js 是主程序.

整个页面的渲染从其中 ReactDOM.render() 函数开始, 其第一个参数是要渲染的元素, 第二个参数是将渲染的结果插入到页面的哪一个容器中.

基本思想

React 中的组件都可以被简化为一个函数的定义. 函数的输出是一串 HTML 标签, 即组件最终渲染的结果, 可以通过组合其他的组件或者 HTML 标签来创建更加复杂的组件.
