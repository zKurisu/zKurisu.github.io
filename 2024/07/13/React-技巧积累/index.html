Hook

以 use 开头的函数被称为 Hook. 如 useState 就是 React 提供的一个内置 Hook.

Hook 是 React 的特性, 因此与普通 JavaScript 函数有区别:

  1. Hook 只能在 React 函数组件或自定义 Hook 中调用
  2. 只能在顶层调用 Hook, 即不要在循环, 条件判断或嵌套函数中调用 Hooks
  3. 自定义 Hook 必须以 use 开头
