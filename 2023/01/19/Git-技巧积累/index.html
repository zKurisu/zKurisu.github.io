Git submodule

Git submodule 允许在一个 Git 仓库中嵌套另一个独立的 Git 仓库.

需要编写 .gitmodules 文件进行配置, 基本语法结构为:

  • submodule 声明一个自定义名称
  • update, 默认是 checkout, 还可以是 rebase, merge, none 等

Git stash

Git stash 用于临时保存工作目录和暂存区的修改, 而无需提交到历史版本. 一般用在, 需要切换分支但不想提交当前半成品代码, 突然要修复紧急 bug, 但当前工作未完成等情况.
