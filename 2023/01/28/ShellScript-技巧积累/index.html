! 系列命令

  • !!, 执行上一条命令
  • !n, 执行历史记录中的第 n 条命令
  • !string, 执行最近的以指定字符串开头的命令

常见特殊变量

  • $0, 脚本名
  • $1,$2… 脚本参数
  • $@, 以数组形式包含全部参数
  • $#, 参数长度
  • $?, 最后一个命令的运行状态
  • $$, 当前脚本的进程 ID
  • $!, 最后一个在后台运行的进程的进程 ID
  • $*, 以字符串形式包含全部参数
  • $IFS, 内部字段分隔符
