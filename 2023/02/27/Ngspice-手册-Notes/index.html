2.1.1 输入文件结构

有两行比较重要:

  • 第一行, 必须是标题, 用注释标识
  • 最后一行, 必须是 .end 加上一个换行符

.include includefile 这一行可以放在 input file 的任意位置.

2.1.2 语法检查

2.1.2.1 有效的 utf-8 字符

读取到无效的 utf-8 字符就会停止读取.

2.1.2.2 特殊字符开头的行

如果一行以:

=[]?()&%$\"!:

开头.

ngspice 会将其替换为 *, 然后报出 warning. 如果设置了 set strict_errorhandling 则会强制退出.
