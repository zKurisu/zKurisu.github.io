第1章-信号与系统

数学上, 信号 可以表示为一个或者多个变量的函数.

信号分类:

  • 连续时间信号, 用 t 表示, 且用 () 包裹, 如 x(t)
  • 离散时间信号, 用 n 表示, 且用 [] 包裹, 如 x[n]

有些离散时间信号是通过对连续时间信号采样而得到的.

1.1.2 信号能量与功率

很多信号与物理系统中的功率和能量有关.

如电流 i(t) 和 电压 v(t), 这两个信号:

$$
\displaylines
{
\begin{aligned}
p(t) = v(t) i(t) = \frac{1}{R} v^2(t) \newline~ \newline
\int_{t_1}^{t_2}p(t) dt = \int_{t_1}^{t_2} \frac{1}{R} v^2 (t) dt
\end{aligned}
}
$$

把信号视为复数值时, 定义一段时间内信号的总能量:

  • 连续时间信号 x(t)
    $$
    \displaylines
    {
    \begin{aligned}
    \int_{t_1}^{t_2} \left\vert x(t) \right\vert ^2 dt
    \end{aligned}
    }
    $$

  • 离散时间信号 x[n]
    $$
    \displaylines
    {
    \begin{aligned}
    \sum_{n=n_1}^{n_2} \left\vert x[n] \right\vert ^2
    \end{aligned}
    }
    $$

三种重要信号 :

  • 信号具有有限的总能量
  • 信号具有平均功率 $p_{\infty}$ 有限 (意味着总能量无限)
  • 信号的平均功率 $p_{\infty}$ 和总能量 $E_{\infty}$ 都不是有限

周期信号

特点:
$$
\displaylines
{
\begin{aligned}
x(t) = x(t + T)
\end{aligned}
}
$$

这里的 最小正值 T 被称为 $x(t)$ 的 基波周期 $T_0$

除 $x(t)$ 为常数外, 基波周期 都有定义. (为常数时, 基波周期可以为任意值)

一个信号不是周期的 (periodic) 就被称为 非周期信号 (aperiodic)

对于离散时间信号则为:
$$
\displaylines
{
\begin{aligned}
x[n] = x[n + N]
\end{aligned}
}
$$
