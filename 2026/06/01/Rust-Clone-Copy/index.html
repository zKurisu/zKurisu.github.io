Copy

Copy trait 的定义是空, 其只是一个标记 trait, 没有方法需要实现. 它是一个编译期约定, 表示这个类型可以按位复制 (memcpy), 会在编译器在编译器进行检查.

对于 Copy 类型, 赋值操作符 = 执行的是按位拷贝, 不会进行所有权的转移. 需要注意, 只实现了 Clone 的话还是会进行所有权转移.

标准库中有很多关于 Copy 的默认实现, 比如:

  • 不可变引用 &T 永远是 Copy, 即赋值了一个指针, 指向原来的值
  • 可变引用 &mut T 只能 Clone (定义如何手动复制), 而不能 Copy, 因为同时只能有一个可变引用
