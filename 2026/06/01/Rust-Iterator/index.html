Rust-Iterator

介绍

这部分用一个示例说明 Rust 中的 IntoIterator, Iterator 以及他们与 for 循环的关系.

  • IntoIterator trait 负责用 into_iter 方法返回一个实现了 Iterator 的迭代器
  • Iterator trait 定义了 next 方法表明如何进行迭代取值

在 Rust 中:

  • for item in collection, 会自动调用 collection.into_iter()
  • for item in &collection, 会自动调用 (&collection).into_iter()
  • for item in &mut collection, 会自动调用 (&mut collection).into_iter()

Iterator

  • next 函数负责取下一个值, 如果有, 则包裹在 Some 中; 如果没有, 则返回 None
  • 实现了 next 之后, count, last 这些函数就会被默认自动实现

  • into_iter 方法返回的类型需要实现了 Iterator trait
  • Item = Self::Item, 表示 Iterator 包裹的 Item 需要与 IntoIterator 的一致
  • 需要注意这里是 self, 会消耗式获取所有权
