包描述文件

包描述文件 (package description file) 是一个配置文件, 用来描述一个软件包的元数据和构建规则. 比如包的名字, 作者, 版本, 包含哪些模块和文件, 有哪些依赖, 如何编译和构建等.

在 Haskell 生态中, 原生的包描述文件是 .cabal 文件, 但该文件描述的内容很冗长, 语法不够直观, 难以维护.

Hackage 和 Stackage

Hackage (the Haskell Package Repository) 是一个 Haskell packages 的仓库, 每个包用一个 name 和 version number 来标识. 这个仓库包含了所有包的所有版本:

Hackage 的缺点在于包之间可能不兼容, 没有版本兼容性保证.

Stackage 也是一个仓库, 其包含的是兼容版本的 packages 组合:

Haddock

Haddock 是一个文档生成工具, 能根据 annotated Haskell code 来生成 web page 和其他文档, 这些文档信息会在 Hackage 和 Stackage 上显示.
