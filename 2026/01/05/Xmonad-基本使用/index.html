基本使用

默认键位

默认的 Modify Key 是 Alt 键.

  • mod-shift-reture 打开一个 xterm
  • mod-space 切换 tilling algorithm
  • mod-comma (逗号 ,) 和 mod-period (句号 .) 来减少 master pane 部分的 window 数量
  • mod-return, mod-shift-j, mod-shift-k 切换 focused window
  • mod-h, mod-l 调整 window 大小
  • mod-button1 (鼠标左键) 拖出一个 window 成为 floating window
  • mod-button3 (鼠标右键) 为拖出的一个 window 调整 window 大小
  • mod-t 将 floating window 放回
  • mod-shift-c 关闭一个 window
  • mod-shift-q 退出 xmonad
  • mod-p 打开 dmenu
  • mod-1 到 mod-9 切换 work space
  • mod-q, 重新加载配置文件

X11 规范

这里介绍两个概念:

  • ICCCM compliant, 指符合 Inter-Client Communication Conventions Manual. 这个是 X11 的传统标准, 制定于 1988 年, 定义了窗口间的通信, 焦点管理, 剪切板等基础协议, 提供了基本的窗口管理功能
  • EWMH compliant, 指符合 Extened Window Manager Hints. 这是先到 X11 扩展标准, 制定于 1999 年, 扩展了 ICCCM, 增加任务栏, 虚拟桌面, 工作区管理等. 被 GNOME, KDE, XFCE 等现代桌面环境采用

XMonad 可以引入下面模块来兼容 EWMH:

主要考虑两个函数:

  • ewmh, 添加 EWMH 工作区支持, 让 xmonad 能与面板/任务栏通信
  • ewmhFullscreen, 添加 EWMH 全屏支持, 让 firefox, libreoffice 等软件可以主动请求全屏, 然后按 Esc 退出
