Apache-Arrow-基本使用

Apache Arrow 项目地址

介绍

Apache Arrow 是一个跨语言的项目, 其定义了一套用于内存数据的列式内存格式.

在以往, Python (Pandas), Spark, C++ 之间传输数据需要不断的 “序列化” 和 “反序列化”, 这会非常耗时. Arrow 提供了一套标准, 让不同系统可以直接读取同一块内存, 并且读取数据时不需要移动或复制数据, 速度极快.

PyArrow

PyArrow 是 Apache Arrow 的 Python 实现版本, 便于 Python 读取内存.

Getting Started

PyArrow Getting Started 文档

Arrow 中数据的基本单位是 column, 而一个 pyarrow.Array 就是一列相同类型的数据, 其在内存中物理连续, 并且不可变.

而 pyarrow.Table 是由多个 Array 组合而成的, 每个 Array 对应一列, 并且每一列都有一个名字 (Schema).

Table 的每一列数据类型可以不同, 但行数必须相同.

这里先介绍下 Parquet, 其是一种列式存储文件格式, 可以对比 CSV 来说明:

  • CSV 是行式存储: 适合人类阅读, 但如果查询某几列时需要读取整行
  • Parquet 是列式存储: 当查询列时只需要读取需要的列, 压缩率极高, 读取速度极快
