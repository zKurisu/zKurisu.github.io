Abseil Python 官网

介绍

Abseil Python 是 Google 开发并维护的开源代码库, 是用于构建高性能, 生产级 AI 模型和复杂应用的基础工具集. 其提供比标准库更好的参数解析, 日志系统, 测试框架, 并提供一个标准入口管理 APP 的生命周期.

QuickStart

  • flags.DEFINE_xx 函数的第一个参数是 arg_name, 第二个是 default_value, 第三个是 desc_msg

Flag types

absl.flags 中定义了一系列 DEFINE_* 函数, 来定义不同类型的 arguments, 这些函数的参数格式是一样的, 分别接收 name, default value, help-string, 和 optional ‘short’ name.

这里列出一些函数, 具体的查看 absl flags 文档:

  • DEFINE_string, 获取任意输入并转换为 string
  • DEFINE_bool or DEFINE_boolean, 通常建议不接受输入, --myflag 这种形式会将 FLAGS.myflag 设置为 True; --nomyflag 这种形式会把 FLAGS.myflag 设置为 False; 手动传递的方式为 --myflag=true 和 --myflag=false
  • DEFINE_float, 将输入转换为 floating point number. 该函数可以接受两个额外的参数 lower_bound 和 upper_bound
  • DEFINE_integer, 将输入转换为 integer. 同样可以传入额外参数 lower_bound 和 upper_bound

实现细节

DEFINE_* 会创建一个 Flag 对象, 然后将其注册到 FlagValues 对象中 (就是 FLAGS, 定义在 __init__.py 中).

FlagValues 对象会扫描 command line arguments 然后将参数传入到对应的 Flag 对象中, Flag 对象会检查类型并做类型转换, 之后转换后的值会保存为 FlagValues 的 attributes.
