Haskell-序列化

介绍

序列化是指将数据结构转为字节流. Haskell 的 Data.Binary 模块包含 binary serialisation 的相关内容, 主要是将 Haskell value 转换为 ByteString (以及从 ByteString 转换回来).

Binary typeclass

需要进行序列化的类型需要实现 Binary typeclass, 其提供 put 和 get 两个函数, 分别用于 encode 和 decode 一个 Haskell value 到 lazy ByteString.

  • put 将一个 value 编码到 Put monad 中
  • get 将一个 value 解码到 Get monad 中
  • putList 将 a list of values 编码到 Put monad 中

Put 定义为 type Put = PutM (), PutM 是一个类型构造器, () 表示空类型.

newtype PutM a 也实现了这些 typeclass: Applicative, Functor, Monad, Monoid, Semigroup.

Put Monad 描述了如何将数据逐步写入到二进制格式. runPut :: Put -> ByteString 用于将其转换为 ByteString 类型.

data Get a 实现了这些 typeclass: MonadFail, Alternative, Applicative, Functor, Monad, MonadPlus. runGet :: Get a -> ByteString -> a 通常用于将一个 ByteString 解析为具体的 Haskell value.

大部分类型都实现了 Binary typeclass, 比如常见的 Int, String, Bool 等等.
