介绍

ResNet (Residual Network, 残差神经网络), 利用 residual connection 和 shortcut connection, 解决了深度学习中的 vanishing gradient 问题, 使得训练成百上千层深度的网络成为可能.

在不使用 residual 时, 训练层数越多会导致结果更差 (过拟合, 梯度消失/爆炸等):

重要概念

Residual

$$
\displaylines
{
\begin{aligned}
F(x) = H(x) - x
\end{aligned}
}
$$

  • $F(x)$ 就是残差函数
  • $H(x)$ 是目标输出函数
  • $x$ 是输入

Identity Mapping

Identity mapping, 恒等映射, 指一个函数的输出等于其输入, 即 $f(x) = x$.

在神经网络中, 需要通过一条没有任何权重, 直接跳转的连接来实现.

CIFAR10

CIFAR10, Canadian Institute for Advanced Research 10, 是一个用于图像识别和机器学习的小型数据集. 包含:

  • 10 个类别的彩色图片
  • 60,000 张图像: 50,000 张用于训练 (每个类别 10,000 张), 10,000 (每个类别 1,000 张) 用于测试
  • 每张图片为 32x32 像素

VGG

VGG, 是一个经典的用于图像分类任务的 CNN 模型, 由牛津大学 Visual Geometry Group 提出, 因此得名. 最著名的版本是 VGG-16 和 VGG-19, 其中数字代表层数.

之前的网络, 如 AlexNet, 都会使用较大的卷积核, 如 11x11, 7x7, 5x5. 而 VGG 的设计中都使用 3x3 卷积核.

整体架构

  • $x$ 是浅层的输出
  • weight layer 是新加的层
  • 理想的输出是 $H(x)$
  • 此时不让网络直接去找理想输出 $H(x)$, 而是学习 $F(x) = H(x) - x$, 也就是目标函数与输入 $x$ 的差值
  • $F(x)$ 称残差函数

外围的 shortcut connection 其实就是残差连接, 让输入直达输出. 此时网络的输出从 $h(x) = f(wx + b)$ 改写成 $h(x) = f(x) + x$.

Shortcut connection 是解决 vanishing gradient 的关键:

  • 没有 shortcut connection 的传统网络, 梯度必须从输出层一层一层地反向传播到输入层, 每经过一层, 梯度就可能被减弱一次, 层数越深, 梯度在传播到底层之前就可能变得无限接近于零 (消失) 或无限大 (爆炸).

对于传统网络, 第 l 层的输出是:

$$
\displaylines
{
\begin{aligned}
x_l = F(x_{l-1})
\end{aligned}
}
$$

最终输出 $y$ 相对于早期输入 $x_s$ 的梯度为:

$$
\displaylines
{
\begin{aligned}
\frac{\partial y}{\partial x_s} = \frac{\partial y}{\partial x_L} \cdot \frac{\partial x_L}{\partial x_{L-1}} … \frac{\partial x_{s+1}}{\partial x_s}
\end{aligned}
}
$$

  • 有 shortcut connection 的网络, 创建了一条从早期层直接连接到后期层的通道, 梯度可以通过这条捷径几乎无衰减地直接反向传播到较浅的层, 避免梯度消失.

对于带有 shortcut connection 的网络, 第 l 层的输出是:

$$
\displaylines
{
\begin{aligned}
x_l = x_{l-1} + F(x_{l-1})
\end{aligned}
}
$$

此时计算 $x_l$ 对 $x_{l-1}$ 的梯度:

$$
\displaylines
{
\begin{aligned}
\frac{\partial x_l}{\partial x_{l-1}} = \frac{\partial}{\partial x_{l-1}} [x_{l-1} + F(x_{l-1})] \newline~ \newline
= I + \frac{\partial F(x_{l-1})}{\partial x_{l-1}}
\end{aligned}
}
$$

其中 $I$ 是单位矩阵.

此时反向传播梯度的计算可以是:

$$
\displaylines
{
\begin{aligned}
\frac{\partial L}{\partial x_{l-1}} = \frac{\partial L}{\partial x_l} \cdot (I + \frac{\partial F(x_{l-1})}{\partial x_{l-1}})
\end{aligned}
}
$$

此时梯度 $\frac{\partial L}{\partial x_{l-1}}$ 有两条路可走:

  • 一条是通过复杂的非线性函数 $F$, 梯度为: $\frac{\partial L}{\partial x_l} \cdot \frac{\partial F(x_{l-1})}{\partial x_{l-1}}$
  • 另一条是通过 identity mapping 的路径, 梯度为: $\frac{\partial L}{\partial x_l} \cdot I$

此时, 就算函数 $F$ 的梯度非常小, 也总会有 $\frac{\partial L}{\partial x_l}$ 传回去.

Residual Block 实现

由于 residual block 的输出计算为:

$$
\displaylines
{
\begin{aligned}
output = F(x) + x
\end{aligned}
}
$$

需要保证 F(x) 和 x 的维度一致, 而这里的 downsampling 的作用就是对 x 进行维度调整. 比如:

  • 1x1 的卷积不改变空间尺寸, 仅调整通道数
  • BatchNorm2d 保持数值稳定
