Perceptron-network

介绍

Perceptron, 感知机, 是最简单的神经网络模型, 用于二分类问题, 其核心是一个线性分类器. 不能拟合 XOR 以及非线性问题:

(XOR 示例)

“Perceptron” 在英文中来自 “Perception” (感知) 和 “-tron” (装置/工具), 想表达模拟人脑神经元通过感官接收信息并理解世界的能力.

该模型的局限在与无法处理非线性关系.

模型结构

结构如下:

  • Input layer, 接收特征向量 $x = (x_1, x_2, …, x_n)$
  • Weight and bias, 每个输入对应一个权重 $w_i$, 外加偏置 $b$
  • Activation function, 使用 step function (阶跃函数), 输出 0 或 1

数学模型如下:

$$
\displaylines
{
\begin{aligned}
y = f(\sum_{i=1}^n w_i x_i + b), f(z) = \begin{cases}
1,\ if\ z \ge 0, \newline~ \newline
0,\ otherwise
\end{cases}
\end{aligned}
}
$$

  • $y \in \{0, 1\}$ 是输出的分类结果
  • 目标是学习到一组权重 $w$ 和偏置 $b$, 使得分类错误最小

算法步骤

  1. 初始化: 权重 $w$ 和偏置 $b$ 设为随机值或 0
  2. 迭代更新: 对每个训练样本 (x, d) (d 为真实标签):
    a. $w \leftarrow w + \eta (d - y)x, b \leftarrow b + \eta (d - y)$
    b. $\eta$ 是学习率, 控制更新的步长
    c. $d-y$ 是误差信号, 0 表示分类正确, 不更新
    d. 损失函数为 $\phi (y, x, w) = max(0, -y<w,x>)$
  3. 终止条件: 所有样本类别分类正确或达到最大迭代次数

Multilayer Perceptron, MPL

Multilayer Perceptron, 多层感知机, 是最基础的前馈神经网络, 有多个全连接层 (fully connected layers) 堆叠而成, 能够解决复杂的非线性分类和回归问题.

其相比单层感知机, 引入了 sigmoid, relu 等非线性激活函数, 用反向传播来更新参数.

基本结构

MPL 由 3 层或更多层神经元组成:

  • Input layer, 输入层, 接收原始数据 $x \in R^n$ (如像素, 特征向量)
  • Hidden layer, 1 层或多层非线性变换层, $W_1 \in R^{m \times n}$, $b_1 \in R^m$
  • Output layer, 输出层, 参数 $w_2 \in R^m$, $b_2 \in R$, 生成最终预测结果 (如分类概率, 回归值)

数学原理

假设只有 1 个隐藏层.

从输入层到隐藏层有:

$$
\displaylines
{
\begin{aligned}
h = f(W_1 x + b_1)
\end{aligned}
}
$$

  • $W_1$ 是输入层到隐藏层的权重矩阵
  • $b_1$ 是隐藏层的 bias
  • $f$ 是激活函数 (比如 $f(z) = max(0,z)$)

从隐藏层到输出层:

$$
\displaylines
{
\begin{aligned}
y = g(W_2 x + b_2)
\end{aligned}
}
$$

  • $g$ 是输出层的激活函数 (比如 softmax)

反向传播时利用梯度下降进行参数更新.
