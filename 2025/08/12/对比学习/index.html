介绍

Contrastive learning, 对比学习, 是一种机器学习方法, 核心思想是让模型学习区分相似 (正样本) 和不相似 (负样本) 的数据. 其通过构造对比损失函数, 使相似样本在特征空间中靠近, 不相似样本相互远离.

这里的特征空间通常指可学习的 embedding space. 正样本间的 embedding 在空间中相近.

Contrastive learning 通常分为:

  • Supervised Contrastive Learning
  • Self-Supervised Contrastive Learning

相关概念

正负样本

正样本, Positive samples, 指我们感兴趣的目标, 含感兴趣特征/模式的样本.

负样本, Negative samples, 指不包含目标特征, 代表背景, 正常情况或非目标类别的样本.

在实际应用中, 通常正样本比负样本少得多.

Supervised Contrastive Learning

Supervised Contrastive Learning (SupCon) 利用标注的数据 (能更精确定义谁是正样本, 谁是负样本) 来训练模型, 让其能够区分正样本和负样本. 让正样本在特征空间中相互靠近, 让负样本远离正样本的空间.

在 SupCon 中:

  • 正样本指, 与 anchor 样本属于相同类别的其他样本. 比如一张猫的图片 A, 其他猫的图片 B, C, D 都是它的正样本 (如果它们标签一样).
  • 负样本指, 与 anchor 样本属于不同类别的其他样本, 比如一张猫的图片 A, 狗的图片 E, F 就是它的负样本

通常会让一个 batch 内所有同标签样本作为正样本, 不同标签作为负样本.

Loss 的简化公式如下, 假设 anchor 样本 i, 特征是 $z_i$, 正样本为 $j \in P(i)$, 负样本为 $k \in N(i)$:

$$
\displaylines
{
\begin{aligned}
L_i = - \frac{1}{|P(i)|} \sum_{j \in P(i)} log \frac{exp(sim(z_i, z_j)/\tau)}{\sum_{k \in \{i\} \cup N(i)} exp(sim(z_i,z_k)/\tau)}
\end{aligned}
}
$$

  • $sim(z_i, z_j)$ 是余弦相似度
  • $\tau$ 是温度超参数
  • $P(i)$ 是所有与 anchor $i$ 同标签的样本集合
  • $N(i)$ 是所有与 anchor $i$ 不同标签的样本集合

Self-Supervised Contrastive Learning

Self-Supervised Contrastive Learning (SSCL) 从未显式标注的数据中学习表征, 从而区分正负样本.

SSCL 依赖于 pretext tasks (先验任务) 从 unlabled data 中创建正负样本对. Pretext tasks 需要从数据中捕获 meaningful features 和 similarities.

一个常用的 pretext task 是创建 augmented views, 把同一 instance 创建出的 augmented views 作为正样本集, 把其他 instance 创建出的 augmented views 作为负样本集, 然后用于模型训练.
