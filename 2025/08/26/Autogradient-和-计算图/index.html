介绍

详细说明下计算图以及如何自动计算梯度.

梯度的简单例子

假设有: $f(x,y) = xy$, 分别计算两个输入的偏导:

$$
\displaylines
{
\begin{aligned}
f(x,y) = xy \rightarrow \frac{\partial f}{\partial x} = y, \frac{\partial f}{\partial y} = x
\end{aligned}
}
$$

求导的本质就是求变量变化 $h$ 时 (视作直线) 的斜率 :

$$
\displaylines
{
\begin{aligned}
\frac{d f(x)}{dx} = \underset{h \rightarrow 0}{lim} \frac{f(x+h) - f(x)}{h}
\end{aligned}
}
$$

上述式子可重写为:

$$
\displaylines
{
\begin{aligned}
f(x + h) = f(x) + h \frac{df(x)}{dx}
\end{aligned}
}
$$

梯度 (gradient), 就是各偏导组成的向量: $\nabla f = [ \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} ] = [y, x]$. 此时, 若求 $x$ 的 partial derivative 可以称 “the gradient on x”.

复合函数求导例子

复合函数需要用 chain rule 求导. 一个简单例子:
$$
\displaylines
{
\begin{aligned}
f(x,y,z) = (x+y)z \newline~ \newline
q = x + y, f = qz
\end{aligned}
}
$$

有:

$$
\displaylines
{
\begin{aligned}
\frac{\partial f}{\partial q} = z \newline~ \newline
\frac{\partial q}{\partial x} = 1, \frac{\partial q}{\partial y} = 1
\end{aligned}
}
$$

此时, 求 $\frac{\partial f}{\partial x}$ 可以写为:
$$
\displaylines
{
\begin{aligned}
\frac{\partial f}{\partial x} = \frac{\partial f}{\partial q} \times \frac{\partial q}{\partial x}
\end{aligned}
}
$$

同理 $\frac{\partial f}{\partial y}$ 可以写为:
$$
\displaylines
{
\begin{aligned}
\frac{\partial f}{\partial y} = \frac{\partial f}{\partial q} \times \frac{\partial q}{\partial y}
\end{aligned}
}
$$

写成代码的形式:

此时的梯度可以写为 [dfdx, dfdy, dfdz].

Conputational graph

上述计算可以绘制成一个 graph:

  • forward pass 指从左到右的计算过程
  • backward pass 指从右向左利用 chain rule 计算梯度的过程

Circuit diagram 中每个节点在得到输入时, 可以进行两个计算:

  1. 计算 output value
  2. 计算该 output 于 input 的 local gradient

也就是说, forward pass 不仅会计算 output, 还会计算 local gradient 并缓存.

在 forward pass 结束后, backward pass 可以利用 chain rule 将各 local gradient 乘起来得到总输出对于所有 inputs 的 gradient.

还是以上面图为例子, forward pass 时, 先进行的计算是:
$$
\displaylines
{
\begin{aligned}
q = x + y, x = -2, y = 5 \newline~ \newline
\therefore q = 3, \frac{\partial q}{\partial x} = 1, \frac{\partial q}{\partial y} = 1 \newline~ \newline
\end{aligned}
}
$$

然后进行的计算是:
$$
\displaylines
{
\begin{aligned}
f = q z, q = 3, z = -4 \newline~ \newline
\therefore f = -12, \frac{\partial f}{\partial q} = z = -4, \frac{\partial f}{\partial z} = q = 3
\end{aligned}
}
$$

之后进行 backward pass, 要求:

$$
\displaylines
{
\begin{aligned}
\frac{\partial f}{\partial q} &= -4 (forward pass 求出了) \newline~ \newline
\frac{\partial f}{\partial z} &= 3 (forward pass 求出了) \newline~ \newline
\frac{\partial f}{\partial x} = \frac{\partial f}{\partial q} \times \frac{\partial q}{\partial x} &= -4 \times 1 = -4 (两个都在 forward pass 过程中求出了) \newline~ \newline
\frac{\partial f}{\partial y} = \frac{\partial f}{\partial q} \times \frac{\partial q}{\partial y} &= -4 \times 1 = -4 (两个也都在 forward pass 过程中求出了) \newline~ \newline
\end{aligned}
}
$$

Sigmoid example

以 sigmoid 函数作为复杂一点的例子:

$$
\displaylines
{
\begin{aligned}
f(w, x) = \frac{1}{1 + e^{-(w_0 x_0 + w_1 x_1 + w_2)}}
\end{aligned}
}
$$

这里给出一些常见的求导公式:
$$
\displaylines
{
\begin{aligned}
f(x) = \frac{1}{x} &\rightarrow \frac{df}{dx} = - \frac{1}{x^2} \newline~ \newline
f_c(x) = c + x &\rightarrow \frac{df}{dx} = 1 \newline~ \newline
f(x) = e^x &\rightarrow \frac{df}{dx} = e^x \newline~ \newline
f_a(x) = ax &\rightarrow \frac{df}{dx} = a
\end{aligned}
}
$$

  • 输入是 [x_0, x_1], 可学习的 weights 是 [w_0, w_1, w_2]

手动求导:

$$
\displaylines
{
\begin{aligned}
\sigma (x) = \frac{1}{1 + e^{-x}} \newline~ \newline
\rightarrow \frac{d \sigma(x)}{dx} = \frac{e^{-x}}{(1 + e^{-x})^2} = ( \frac{1 + e^{-x} - 1}{1 + e^{-x}}) ( \frac{1}{1 + e^{-x}}) = (1 - \sigma(x)) \sigma(x)
\end{aligned}
}
$$
