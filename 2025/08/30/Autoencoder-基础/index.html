Autoencoder-基础

介绍

AE, Autoencoder, 是一种特殊的神经网络, 用于学习数据的表示. 其思想是通过无监督学习来压缩和重建数据.

用一组输入数据来训练 Autoencoder, 输出要尽可能与输入相同, 此时才说明其掌握了重构的能力.

VAE

VAE, Variational Autoencoder, 变分自编码器, 是一种结合了深度学习和概率图模型的生成式人工智能方法. 其为简单的 encoder-decoder 结构, 但包含概率生成功能.

对于传统的 autoencoder 而言, encoder 输出的 “中间表示” 没有明确的概率意义:

而 variational autoencoder 中, 能够学习数据的概率分布以生成新样本, 且 “中间表示” 具有良好的概率结构和连续性:

  • 此时编码器的输出不再是 “潜在向量” (中间表示), 而是用于计算高斯分布的两个参数向量:
    • 均值 $\mu$ 用于表示输入数据分布的中心
    • 方差 $\sigma$ 用于表示输入可以有多大的变化或不确定性

相关概念

Log-variance

Log-variance, 对数方差, 指方差 $\sigma^2$ 取自然对数 $\ln$ 后的值:

$$
\displaylines
{
\begin{aligned}
logvar = \ln(\sigma^2)
\end{aligned}
}
$$

让神经网络直接输出 $\sigma^2$ 会有一些问题, 但 logvar 都能很好解决:

  1. 确保方差为正, 在梯度下降过程中, 可能会把 $\sigma^2$ 调整为负数, 导致这是一个无意义的方差值 (比如 $\sigma^2 = 0.01,lr=0.1,\nabla=0.2, 导致 \sigma -= lr * \nabla = -0.01$). 而不论 logvar 取什么值, $\sigma^2 = exp(logvar)$ 得到的方差永远为正
  2. 数值稳定, 当方差非常小时, $\sigma^2$ 可能进入浮点数精度下限导致下溢为 0, logvar 在对数空间中计算, 数值范围更宽, 当方差很小时, logvar 也是一个很大的负数, 数值计算安全 (虽然也会溢出, 但是从比平方稳定)
  3. $\sigma^2$ 的变化范围是 $(0,+\inf)$, 而 logvar 的取值范围是 $(-\inf, +\inf)$, 无约束 (如果直接输出 $\sigma^2$, 需要加上 relu 约束丢弃小于零的值)

采样

VAE 的采样分两步:

  1. 从标准正态分布 (均值为 0, 方差为 1) 中随机采样一个噪声向量 $\epsilon$:

$$
\displaylines
{
\begin{aligned}
\epsilon ~ N(0, 1)
\end{aligned}
}
$$

  1. 通过 reparameterize trick (一个简单的变换), 将随机噪声 $\epsilon$ “推到” 由 $\mu$ 和 $\sigma$ 定义的分布中:

$$
\displaylines
{
\begin{aligned}
z = \mu + \sigma \odot \epsilon
\end{aligned}
}
$$

  • $\odot$ 表示逐元素相乘

分为两步的原因

直接从 $N(\mu, \sigma^2)$ 采样, 由于 $z ~ N(\mu, \sigma^2)$ 不可导, 梯度无法在采样后反向传播回编码器.

使用 reparameterize trick 后, 采样过程被分解为一个可导的确定部分 $\mu + \sigma \cdot …$ 和一个随机部分 $\epsilon$, 梯度可以正确反向传播.

为什么采样 $z ~ N(\mu, \sigma^2)$ 不可导

因为在数学上, 随机性具有不可微性. 直接采样是一个随机过程, 而不是数学函数:

  • 因为对于固定的 (mu, sigma), 每次调用都会得到不同的随机结果
  • 而导数需要衡量 “输入微小变化时输出的变化”, 在随机性下该关系不成立

因此在梯度反向传播时, 在采样处会断裂:

证明: 如果 $\epsilon ~ N(0,1)$, 那么 $z = \mu + \sigma \cdot \epsilon ~ N(\mu, \sigma^2)$

  • 几个符号说明:
    • $E[\epsilon]$ 是 $N(0,1)$ 的均值, 即 0
    • $Var[\epsilon]$ 是 $N(0,1)$ 的方差, 即 1
    • $E[z]$ 是 $N(\mu,\sigma^2)$ 的均值, 即 $\mu$
    • $Var[z]$ 是 $N(\mu,\sigma^2)$ 的方差, 即 $\sigma^2$
    • 也就是说 $E()$ 表示求均值的函数, $Var()$ 表示求方差的函数

推导:
$$
\displaylines
{
\begin{aligned}
E[z] = E[\mu + \sigma \cdot \epsilon] = \mu + \sigma \cdot E[\epsilon] = \mu \newline~ \newline
Var[z] = Var[\mu + \sigma \cdot \epsilon] = \sigma^2 \cdot Var[\epsilon] = \sigma^2 \cdot 1 = \sigma^2
\end{aligned}
}
$$

VAE loss

$$
\displaylines
{
\begin{aligned}
L = E_{q(z|X)} [\log_p (X|z)] + \beta \cdot D_{KL} (q(z|X)\Vert p(z))
\end{aligned}
}
$$

  • 前一项是重建损失:
    • $q(z|X) 是编码器输出的分布, 通常是 $N(\mu,\sigma^2)$ 的高斯分布, 学习如何将数据映射到潜在空间
    • $p(X|z)$ 是解码器的似然, 即解码器重建出原始数据 $X$ 的可能性, 学习如何从潜在表示重建数据
    • $E_{q(z|X)}$ 求期望, 即平均值
  • 后一项是正则化项 (KL 散度), 惩罚那些方差 σ 过大或均值 μ 远离 0 的分布, 防止模型退化成普通自编码器:
    • $p(z)$ 是先验分布, 即希望潜在变量 $z$ 服从的分布, 通常是标准正态分布 $N(0,1)$
    • $D_{KL}(… \Vert …)$ 求 KL 散度, 衡量两个概率分布之间的差异或 “距离”. 公式为 $D_{KL}(q \Vert p) = E_q[log^q_p]$. 这里用来惩罚 $q(z|X)$ 与 $p(z)$ 的差异
