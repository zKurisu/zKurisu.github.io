矩阵乘法的导数

在数学上:
$$
\displaylines
{
\begin{aligned}
Y = AX, A \in R^{m \times n}, X \in R^{n \times p} \newline~ \newline
\frac{\partial Y}{\partial X} = A^T, \frac{\partial Y}{\partial A} = X^T
\end{aligned}
}
$$

但注意 backward() 方法只能由标量调用, 即 1x1 的张量. 因此不能直接调用 Y.backward(), 需要是 Y.sum().backward() 等.

$$
\displaylines
{
\begin{aligned}
Y &= AX, A \in R^{m \times n}, X \in R^{n \times p} \newline~ \newline
L &= \sum_i Y_i = \sum_{i=1}^m \sum_{j=1}^p Y_{ij} \newline~ \newline
\frac{\partial L}{\partial X} &= \frac{\partial L}{\partial Y} \cdot \frac{\partial Y}{\partial X} = 1_{m \times p} \cdot A^T \newline~ \newline
\because \frac{\partial L}{\partial Y_{ij}} &= 1, \forall i,j \newline~ \newline
\because 比如: &\frac{Y_{00} + Y_{01} + … + Y_{mp}}{Y_{00}} = 1
\end{aligned}
}
$$

线性回归

$$
\displaylines
{
\begin{aligned}
x = [x_1, x_2,…,x_n]^T \newline~ \newline
w = [w_1, w_2,…,w_n]^T, b \newline~ \newline
y = w_1x_1 + w_2x_2 + … + w_nx_n + b \newline~ \newline
y = <w, x> + b \newline~ \newline
loss = \frac{1}{2n} \sum_{i=1}^n (y_i - \hat{y_i})^2 = \frac{1}{2n} \Vert y - Xw - b \Vert^2
\end{aligned}
}
$$

  • $y$ 是真实值, $\hat{y}$ 是预测值
  • loss 这里用平方损失

可以把 b 融入 x 和 w 以方便求解:
$$
\displaylines
{
\begin{aligned}
X \leftarrow [X\ 1], w^T \leftarrow [w\ b] \newline~ \newline
loss(X,y,w) = \frac{1}{2n} \Vert y - Xw \Vert^2 \newline~ \newline
\frac{\partial loss(X,y,w)}{\partial w} = \frac{1}{n} (y - Xw)^T X
\end{aligned}
}
$$

  • 此时的损失函数是凸函数, 其最优解满足:

$$
\displaylines
{
\begin{aligned}
\frac{\partial loss(X,y,w)}{\partial w} = 0 \newline~ \newline
\frac{1}{n} (y - Xw)^T X = 0 \newline~ \newline
(y - Xw) X^T = 0 \newline~ \newline
yX^T = XX^T w \newline~ \newline
(X^TX)^{-1} (X^TX) w = (X^T X)^{-1} X^T y \newline~ \newline
(X^TX)^{-1} (X^TX) = I = w^* \newline~ \newline
w^* = (X^TX)^{-1} X^Ty
\end{aligned}
}
$$
