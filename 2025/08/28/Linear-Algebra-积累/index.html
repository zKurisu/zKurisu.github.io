Linear-Algebra-积累

对角矩阵

一个 n x n 的方阵 D 被称为对角矩阵, 如果其主对角线以外的所有元素都为零. 即对于矩阵中任意位置的元素 $d_{ij}$ (i 为行索引, j 为列索引), 满足下面条件:

$$
\displaylines
{
\begin{aligned}
d_{ij} = 0, if\ i \ne j
\end{aligned}
}
$$

主对角线上的元素 $d_{ii}$ 可以是非零, 也可以是 0.

通常可以表示为:

  • $d_1, d_2, …, d_n$ 是主对角线上的元素

简单例子

一个 2x2 的对角矩阵:
$$
\displaylines
{
\begin{aligned}
D_2 = \begin{bmatrix}
1 & 0 \newline~ \newline
0 & 5
\end{bmatrix}
\end{aligned}
}
$$

一个 3x3 的对角矩阵:
$$
\displaylines
{
\begin{aligned}
D_3 = \begin{pmatrix}
5 & 0 & 0 \newline~ \newline
0 & 0 & 0 \newline~ \newline
0 & 0 & 8
\end{pmatrix}
\end{aligned}
}
$$

单位矩阵是一种特殊的对角矩阵, 所有主对角线元素均为 1:

$$
\displaylines
{
\begin{aligned}
I_3 = \begin{pmatrix}
1 & 0 & 0 \newline~ \newline
0 & 1 & 0 \newline~ \newline
0 & 0 & 1
\end{pmatrix}
\end{aligned}
}
$$

相关性质

  1. 矩阵加法/减法: 两个同阶对角矩阵相加或相减, 结果仍是对角矩阵. 只需将对应位置的对角线元素相加或相减:

$$
\displaylines
{
\begin{aligned}
\operatorname{diag}(a_1, \ldots, a_n) \pm \operatorname{diag}(b_1, \ldots, b_n) = \operatorname{diag}(a_1 \pm b_1, \ldots, a_n \pm b_n)
\end{aligned}
}
$$

  1. 矩阵乘法: 两个同阶对角矩阵相乘, 结果仍是对角矩阵. 只需将对应位置的对角线元素相乘:

$$
\displaylines
{
\begin{aligned}
\operatorname{diag}(a_1, \ldots, a_n) \times \operatorname{diag}(b_1, \ldots, b_n) = \operatorname{diag}(a_1b_1, \ldots, a_nb_n)
\end{aligned}
}
$$

当对角矩阵 $D$ 与任意矩阵 $A$ 相乘时:

  • 左乘: 相当于用 $D$ 的对角线元素分别缩放 $A$ 的各行
  • 右乘: 相当于用 $D$ 的对角线元素分别缩放 $A$ 的各列
  1. 幂运算: 对角矩阵的 k 次幂 (k 为整数) 非常容易计算, 只需将对角线上的每个元素求 k 次幂:

$$
\displaylines
{
\begin{aligned}
(\operatorname{diag}(d_1, d_2, \ldots, d_n))^k = \operatorname{diag}(d_1^k, d_2^k, \ldots, d_n^k)
\end{aligned}
}
$$

  1. 转置: 对角矩阵的转置等于其本身

$$
\displaylines
{
\begin{aligned}
D^T = D
\end{aligned}
}
$$

  1. 行列式: 对角矩阵的行列式等于其主对角线所有元素的乘积:

$$
\displaylines
{
\begin{aligned}
\det(D) = d_1 \times d_2 \times \ldots \times d_n
\end{aligned}
}
$$

  1. 逆矩阵: 如果对角矩阵 $D$ 的每个对角线元素有 $d_i \neq 0$, 则其逆矩阵也是对角矩阵, 且每个对角线元素是原对应元素的倒数

$$
\displaylines
{
\begin{aligned}
D^{-1} = \operatorname{diag}(\frac{1}{d_1}, \frac{1}{d_2}, \ldots, \frac{1}{d_n})
\end{aligned}
}
$$
