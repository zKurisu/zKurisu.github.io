K-means-算法实现

该 blog 仅记录一个课程任务. 大纲是照着提交模板写的.

介绍

K-means 算法用于将 N-dimensional population 分到 k 个集合中. 该算法不能保证每次都能找到 optimal partition (k 的取值, 初始化的位置都会影响).

相关概念或术语

Population

Population 在统计学和信号处理中, 指研究对象的完整数据集合.

Within-class variance

Within-class variance, 类内方差, 衡量同一类别 (或聚类) 内部数据的离散程度, 数学表达为:

$$
\displaylines
{
\begin{aligned}
w^2 (S) = \sum_{i=1}^k \int S_i | z - \mu_i |^2 dp(z)
\end{aligned}
}
$$

  • $\sum_{i=1}^k$ 外层求和, 对所有 k 个聚类求和, 用于得到整体聚类质量的评估, 可以用来比较初始条件不同, 或者不同划分时的好坏
  • $\int_{S_i}$ 指第 i 个聚类内所有元素
  • $z$ 是一个数据点
  • $\mu_i$ 是 $S_i$ 的质心 (均值)
  • $dp(z)$ 是概率测度, 描述数据点在z处的概率密度 (连续) 或概率质量 (离散), $dp(z) = f(z) dz$, $f(z)$ 是概率密度函数. 乘以这个概率是为了加权

Probability Mass Function

Probability Mass Function, 概率质量函数, 描述离散随机变量在各取值点上的概率分布, 记为 $p(z)$, 满足:

$$
\displaylines
{
\begin{aligned}
\sum_{z} p(z) = 1
\end{aligned}
}
$$

Conditional Mean

Conditional Mean, 条件均值, 指在已知某些条件下的随机变量的期望值. 其目的是计算 “均值”, 而不是 “概率”.

数学定义:
$$
\displaylines
{
\begin{aligned}
\mu_S = E[Z | Z \in S] = \frac{\int_S dp(z)}{\int_S dp(z)}
\end{aligned}
}
$$

  • 分子, 是数据点 z 在 $dp(z)$ 作为权的加权和
  • 分母, $S$ 的总概率, 也就是所有权之和

Ordinary Sample Mean

Ordinary Sample Mean, 普通样本均值, 仅表示一组数据的平均值:

$$
\displaylines
{
\begin{aligned}
\overline = \frac{1}{n} \sum_{i=1}^n z_i
\end{aligned}
}
$$

Optimal Information Structure

在信息论, 统计学和机器学习中, Optimal Information Structure (最优信息结构) 指的是在给定约束条件下, 最有效地组织和表示数据的方式, 使得信息的损失最小或任务的性能最优.

Online mean computation

Online mean computation, 在线均值算法, 是一种逐步更新均值的方法, 每次仅用一个新数据点调整当前均值, 而无需存储所有历史数据:

$$
\displaylines
{
\begin{aligned}
\mu_n = \frac{(n-1) \cdot \mu_{n-1} + x_n}{n}
\end{aligned}
}
$$

  • $\mu_n$ 是前 n 个数据的均值
  • $x_n$ 是第 n 个新增数据点

算法原理描述

含每个任务相关的模型, 算法细节描述.

输入一个随机序列 $z_1, z_2,…$ ($z$ 表示一个 n-dimension vector), $z_i \in E_N$, 序列中每个元素相互独立且有一个概率测度 $p$.

给定一个 k-tuple $x = (x_1,x_2,…,x_k)$ (也就是 cluster centroids), $x_i \in E_N, i = 1, 2, …, k$, 且 minimum distance partition $S(x)$ 定义为 $S(x) = {S_1(x), S_2(x), …, S_k(x)}$. 比如 $S_1(x)$ 指一个集合, 包含所有离 $x_i$ 最近的点 (这个点要计算与所有 $x_i$ 的距离并比较), 距离相同则分配给 index 更小的集合.

K-means 通过迭代来逼近解, 初始化时, 前 k 个样本点直接作为初始质心:

$$
\displaylines
{
\begin{aligned}
x_i^1 = z_i, w_i^1 = 1, (i=1,…,k)
\end{aligned}
}
$$

  • $x_i^1$ 表示第 1 次迭代的第 $i$ 个质心
  • $z_i$ 表示第 $i$ 个样本点
  • $w_i^1$ 表示第 1 次迭代的权重, 权重值表示历史样本数

第 n 次迭代时有:
$$
\displaylines
{
\begin{aligned}
x_i^{n+1} = \frac{x_i^n \cdot w_i^n + z_{k+n}}{w_i^n + 1} \newline~ \newline
w_{i}^{n+1} = w_i^n + 1 \newline~ \newline
x_j^{n+1} = x_j^n, w_j^{n+1} = w_j^n, j \ne i
\end{aligned}
}
$$

  • $x_i^{n+1}$ 是新的质心, 值是当前质心与新增样本的加权平均, $x_i^n$ 是一个均值, $w_i^n$ 是样本数量, $x_i^n \cdot w_i^n$ 就是总和, $z_{k+n}$ 是新样本点
  • $x_j^{n+1}$ 和 $w_j^{n+1}$ 的变化表明其他的中心点不变 (毕竟一个新增数据点只会归属到一个 cluster 中, 也只会更新一个中心点)

简单来说就是, 初始化时, 分了 k 个 groups, 每个 group 包含一个 random point. 添加一个 point 到一个 group, 说明该 point 离该 group 最近. 之后会更新对应 group 的 mean 值.
