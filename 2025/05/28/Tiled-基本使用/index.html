Tiled-基本使用

Tiled 官方文档

介绍

Tiled 是一个免费, 开源的 2D 地图编辑器, 支持 Tilemap 和对象层的创建, 并能导出多种格式, 如 JSON, CSV, Lua. 也能进行动画编辑, 碰撞编辑等.

相关概念

Terrain Sets

Terrain Sets, 地形集, 是一个用于自动连接相邻 tile 边缘的高级功能, 适合快速绘制自然地形.

在 Tiled 中, 通过定义 tile 之间的连接规则, 让其能够在绘制时自动选择正确的过渡 tile, 避免手动拼接.

部分界面解释

启动页面

创建 map

  • Orientation, 指定义 map 的网格排列方式和显示风格. 如:
    • orthogonal, 正交, 此时所有单元格都是正方形或矩形
    • isometric, 等距, 单元格以等距角度排列, 通常显示为菱形各自
    • 还有另两个参数

添加 layer

这里有 4 种:

  • Tile layer, 指用网格排列的 tile 拼接的地形, 背景, 装饰等. 适合绘制规则的地形
  • Object layer, 一般放置非 tile 的游戏对象, 如玩家出生点, 敌人, 触发器, 宝箱等. 对象一般自由放置 (不受网格限制), 每个对象可附加自定义属性:
  • Image layer, 放置一张静态的背景图, 直接引用外部图片文件, 不支持 tile 分割, 适合绘制1无需交互的大尺寸背景, 可以设置 opacity, offset 等
  • Group layer, 用于嵌套管理其他图层, 以实现复杂地图的层级分组, 其不包含具体内容, 而是作为一个容器

一般情况下只用得到 Tile layer.

如何用 terrain type 来进行匹配, 从而自动选出相邻的 tile, 取决于不同类别的 terrain set. 有三种类型:

  • Corner set, 可以将 tile 的四个角独立标记为不同地形类型, Tiled 会根据相邻 tile 的 corner 匹配情况来选择过渡 tile
  • Edge set, 将 tile 的四条边独立标记为 terrain type
  • Mixed set, 同时支持 cornet 和 edge 标记, Tiled 会先尝试匹配 edge, 然后匹配 corner
