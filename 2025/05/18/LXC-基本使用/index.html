介绍

LXC, LinuX Container, 是一种轻量级的操作系统级虚拟化技术.

LXC 与 Docker 有一些区别:

特性 LXC Docker
虚拟化级别 操作系统级虚拟化（共享内核） 应用级虚拟化（镜像封装）
用途 运行完整 Linux 系统 运行单个应用或微服务
镜像管理 基于模板（如 ubuntu） 基于分层镜像（Dockerfile）
依赖 直接依赖内核功能 依赖 Docker 引擎
