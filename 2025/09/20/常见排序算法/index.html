常见排序算法

Bubble Sort

Bubble Sort, 冒泡排序, 核心思想是重复遍历列表, 比较相邻元素并交换, 每一轮将最大的元素冒泡到末尾.

时间复杂度:

  • 最好: $O(n)$, 如果列表有序
  • 平均: $O(n^2)$
  • 最坏: $O(n^2)$

空间复杂度: $O(1)$, 原地排序.

Selection Sort

Selection Sort, 选择排序, 核心思想是每次从未排序部分选一个最小 (或最大) 元素, 放到已排序部分的末尾 (注意已排序部分是从开头开始的).

时间复杂度: 最好/平均/最坏都是 $O(n^2)$

空间复杂度是 $O(1)$, 原地排序.

实现示例:

Insertion Sort

Insertion Sort, 插入排序, 核心思想是将每个元素插入到已排序部分的适当位置, 从第二个元素开始.

时间复杂度:

  • 最好: $O(n)$
  • 平均/最坏: $O(n^2)$

空间复杂度 $O(1)$.

实现示例:

Quick Sort

Quick Sort, 快速排序, 核心思想是, 选择一个 pivot (基准), 将数组分为小于 pivot 和大于 pivot 的两部分, 递归排序.

时间复杂度:

  • 最好/平均: $O(n\log n)$
  • 最坏: $O(n^2)$

空间复杂度:

  • 平均: $O(\log n)$, 递归栈
  • 最坏: $O(n)$

Merge sort

Merge sort, 归并排序, 核心思想是将数组分成两半, 分别排序, 然后合并两个有序数组.

时间复杂度:

  • 最好/平均/最坏: $O(n \log n)$

空间复杂度:

  • $O(n)$

Heap Sort

Heap Sort, 堆排序, 核心思想是利用 heap (优先队列) 的性质, 每次取堆顶最大/最小元素, 然后调整堆.

时间复杂度:

  • 最好/平均/最坏: $O(n \log n)$

空间复杂度:

  • $O(1)$, 原地排序
