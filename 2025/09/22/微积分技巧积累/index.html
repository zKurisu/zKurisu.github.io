微积分技巧积累

发现微积分忘完了, 以前的笔记也不在, 这里记一记平时遇到的一些知识点, 参考 “普林斯顿微积分” 以及其他 blog.

齐次

虽然这不是微积分中单有的概念, 但后面求解过程中要遇到.

“齐次” (Homogeneous), 核心含义是 “具有相同比例或结构”, 在不同领域有具体的表现形式.

数学中的齐次

齐次方程 (Homogeneous Equation), 一个方程被称为齐次的, 如果方程中的每一项都具有相同的 “次数” (degree) 或 “比例结构”.

比如:

  • 线性齐次方程: $a_1 x_1 + a_2 x_2 + … + a_n x_n = 0$, 这里所有项都是变量的一次项, 且没有常数项
  • 多项式齐次方程: $x^2 + y^2 - z^2 = 0$, 同样所有项都是二次的, 没有常数项

非齐次方程如: $2x + 3y = 5$, 这里 $2x$ 和 $3y$ 是 1 次, 5 是常数项, 各项比例结构不一致.

注意在微分方程中, “齐次” 的定义有所不同, 对于:

$$
\displaylines
{
\begin{aligned}
L(y) = y’’ + p(x)y’ + q(x)y = f(x)
\end{aligned}
}
$$

  • 齐次方程指: $L(y) = 0$, 即 $f(x) = 0$
  • 非齐次方程指: $L(y) = f(x)$, 即 $f(x) \ne 0$
  • 此时 “齐次性” 不受 $f(x)$ 的次数影响

齐次解 (Homogeneous Solution), 在求解线性微分方程或线性代数方程组时, 齐次解指对应的齐次方程的解 (没有非齐次项时的解). 通常用于表示系统的 “固有行为”.

齐次性 (Homogeneity), 一个函数或系统被称为具有齐次性, 如果其满足 “比例缩放性质”:

  • 一个函数 $f(x_1, x_2, …, x_n)$ 是 $k$ 次齐次函数, 如果对于任意标量, 有:

$$
\displaylines
{
\begin{aligned}
f(\lambda x_1, \lambda x_2, …, \lambdaa x_n) = \lambda^k f(x_1, x_2, …, x_n)
\end{aligned}
}
$$

物理学中的齐次

齐次坐标 (Homogeneous Coordinate), 在计算机图形学和几何学中, 齐次坐标是一种用额外维度表示比例的方法, 例如二维空间中, 点 $(x, y)$ 可以表示为齐次坐标 $(x,y,1)$, 缩放后的点 $(\lambda x, \lambda y)$ 可以表示为 $\lambda x, \lambda y, \lambda)$.

解一阶微分方程

常见的可解的一阶微分方程有:

  • 一阶可分离变量方程
  • 一阶线性方程
  • 线性常系数方程

可分离变量的一阶微分方程

如果能够把一阶微分方程中所有关于 $y$ 的部分 (包括 $dy$) 放在一边, 所有关于 $x$ 的部分放在另一边, 则该微分方程被称为是 “可分离变量的”, 例如:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} = ky
\end{aligned}
}
$$

可以整理为:

$$
\displaylines
{
\begin{aligned}
\frac{1}{ky} dy = dx
\end{aligned}
}
$$

此时对两边进行积分:

$$
\displaylines
{
\begin{aligned}
\int \frac{1}{ky} dy = \int dx \newline~ \newline
\frac{1}{k} ln|y| = x + C
\end{aligned}
}
$$

其中 $C$ 为常数, 要解 $y$ 需要两边乘 $k$ 并取指数:

$$
\displaylines
{
\begin{aligned}
|y| = e^{kx + kC} = e^{kC} e^{kx}
\end{aligned}
}
$$

$\pm e^{kC}$ 是不为零的常数, 称为 $A$, 因此式子简化为:

$$
\displaylines
{
\begin{aligned}
y = Ae^{kx}
\end{aligned}
}
$$

一阶线性方程

一阶线性方程如:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} + p(x)y = q(x)
\end{aligned}
}
$$

  • $p$ 和 $q$ 是关于 $x$ 的函数
  • $y$ 和 $dy/dx$ 的幂次都是 1, 即线性的

这通常不能分离变量, 需要一些技巧, 找到积分因子 $\mu (x)$ 使得方程两边乘以 $\mu (x)$ 后, 左边能凑成某个乘积的导数, 即:

$$
\displaylines
{
\begin{aligned}
\mu (x) \cdot \frac{dy}{dx} + \mu (x) P(x) y = \frac{d}{dx} [\mu (x) y]
\end{aligned}
}
$$

这样的积分因子为:
$$
\displaylines
{
\begin{aligned}
\mu (x) = e^{\int p(x) dx}
\end{aligned}
}
$$

比如:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} + 6 x^2 y = e^{-2x^3} sin(x) \newline~ \newline
\mu (x) = e^{\int 6x^2 dx} = e^{2x^3} \newline~ \newline
e^{2x^3} \frac{dy}{dx} + 6 x^2 e^{2x^3} y = sin(x) \newline~ \newline
\frac{d}{dx} (e^{2x^3 y}) = sin(x) \newline~ \newline
e^{2x^3} y = \int sin(x) dx = -cos(x) + C \newline~ \newline
y = (C - cos(x)) e^{-2x^3}
\end{aligned}
}
$$

常系数微分方程

常系数微分方程形如:

$$
\displaylines
{
\begin{aligned}
a_n \frac{d^n y}{dx^n} + … + a_2 \frac{d^2 y}{dx^2} + a_1 \frac{dy}{dx} + a_0 y = f(x)
\end{aligned}
}
$$

  • 也就是 y 以及 y 的导数前的系数是常量

比如:

$$
\displaylines
{
\begin{aligned}
3 \frac{dy}{dx} - sin(5x) = 12x - 6y
\end{aligned}
}
$$

可以整理为:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} + 2y = 4x + \frac{1}{3} sin(5x)
\end{aligned}
}
$$

  • 这是一阶常系数线性方程, 在这个例子中用积分因子求解会比较困难

一个简单的例子, 假设右边关于 x 的部分为 0:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} - 3y = 0
\end{aligned}
}
$$

  • 这种方程称为 “齐次” 的

解一阶齐次方程

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} + ay = 0
\end{aligned}
}
$$

其就是:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} = ky
\end{aligned}
}
$$

的形式, 其中 $k = -a$, 因此解为:

$$
\displaylines
{
\begin{aligned}
y = Ae^{-a x}
\end{aligned}
}
$$

对于微分方程:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} - 3y = 0
\end{aligned}
}
$$

  • 可直接写出解的形式 $y = Ae^{3x}$, 其中 $A$ 为常数

解二阶齐次方程

对于:

$$
\displaylines
{
\begin{aligned}
a \frac{d^2 y}{dx^2} + b \frac{dy}{dx} + cy = 0
\end{aligned}
}
$$

需要用一种特殊的揭发, 构造 “特征方程” 并求解.

将上述二阶齐次方程写为:

$$
\displaylines
{
\begin{aligned}
ay’’ + by’ + cy = 0
\end{aligned}
}
$$

构造特征二次方程并求解:

$$
\displaylines
{
\begin{aligned}
a t^2 + bt + c = 0
\end{aligned}
}
$$

  • 若有两个不同的实根 $\alpha$ 和 $\beta$, 解为:

$$
\displaylines
{
\begin{aligned}
y = Ae^{\alpha x} + Be^{\beta x}
\end{aligned}
}
$$

  • 若只有一个实根 $\alpha$, 解为:

$$
\displaylines
{
\begin{aligned}
y = Ae^{\alpha x} + B x e^{\alpha x}
\end{aligned}
}
$$

  • 若有两个复根, 它们讲是共轭的, 形如 $\alpha \pm i \beta$, 解为:

$$
\displaylines
{
\begin{aligned}
y = e^{\alpha x} (A cos(\beta x) + B sin(\beta x))
\end{aligned}
}
$$

比如一个特征二次方程: $t^2 - t - 20 = 0$, 解为 $t = -4$ 和 $t = 5$, 则 $y’’ - y’ - 20y = 0$ 的解的形式为:

  • $y = Ae^{-4x} + B e^{5x}$

非齐次方程和特解

当方程右边仅有 $x$ 时:

$$
\displaylines
{
\begin{aligned}
y’’ - y’ - 20y = e^x
\end{aligned}
}
$$

可以猜一个解 $- \frac{1}{20} e^x$ 满足. 这个解 $y_P$ 称 Particular solution (特解). 有:

$$
\displaylines
{
\begin{aligned}
y_P’’ - y_P’ - 20y_P = e^x
\end{aligned}
}
$$

齐次解 $y_H$ 的形式为: $y = Ae^{-4x} + Be^{5x}$, 有:

$$
\displaylines
{
\begin{aligned}
y_H’’ - y_H’ - 20y_H = 0
\end{aligned}
}
$$

相加有:

$$
\displaylines
{
\begin{aligned}
(y_H + y_P)’’ - (y_H + y_P)’ - 20(y_H + y_P) = e^x
\end{aligned}
}
$$

  • 取 $y = y_H + y_P$, 也是该微分方程的一个解, 这也是非齐次方程的所有解的形式

特解通常通过查表得到, 并代回原式求常数项:

比如:

$$
\displaylines
{
\begin{aligned}
y’ - 3y = 5e^{2x}
\end{aligned}
}
$$

查表知道特解形式为 $y_P = Ce^{2x}$, 代回原式:

$$
\displaylines
{
\begin{aligned}
2C e^{2x} - 3Ce^{2x} = 5e^{2x} \newline~ \newline
C = -5
\end{aligned}
}
$$

因此知道 $y_P = -5e^{2x}$.

积分法求特解

当非齐次项 $f(x)$ 的形式较为复杂时, 难以通过查表匹配到标准形式, 积分法通过积分将非齐次项的影响 “累计” 到解中, 从而构造出满足方程的一个特定解.

对于一阶线性微分方程:

$$
\displaylines
{
\begin{aligned}
\frac{dy}{dx} + P(x) y = f(x)
\end{aligned}
}
$$

特解 $y_P$ 的构造方法为:

$$
\displaylines
{
\begin{aligned}
y_p (x) = \int_{x_0}^x e^{- \int_{x_0}^t P(s) ds} f(t) dt
\end{aligned}
}
$$

具体步骤为:

  1. 解齐次方程:

$$
\displaylines
{
\begin{aligned}
y_h(x) = C e^{-\int P(x) dx}
\end{aligned}
}
$$

  1. 常数交易, 设特解为 $y_p (x) = C(x) e^{-\int P(x) dx}$, $C(x)$ 是待定函数
  2. 将特解代入非齐次方程, 化简后有:

$$
\displaylines
{
\begin{aligned}
C’(x) = f(x) e^{\int P(x) dx}
\end{aligned}
}
$$

  1. 积分求 $C(x)$

$$
\displaylines
{
\begin{aligned}
C(x) = \int f(x) e^{\int P(x) dx} dx + D
\end{aligned}
}
$$

  1. 构造特解

$$
\displaylines
{
\begin{aligned}
y_p (x) = e^{-\int P(x) dx} \int f(x) e^{\int P(x) dx} dx
\end{aligned}
}
$$

(不是很懂)
