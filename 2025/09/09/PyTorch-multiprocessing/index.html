PyTorch-multiprocessing

介绍

Python 标准库中的 multiprocessing 模块与 PyTorch 的 Tensor/CUDA 张量 不兼容, 比如:

  • Tensor 不可跨进程直接传递
  • CUDA 上下文问题
  • 共享内存问题

因此使用 torch.multiprocessing, 来兼容.
