Seq2Seq-基础

介绍

Seq2Seq, Sequence-to-Sequence, 是一种将输入序列 (如句子, 时间序列) 映射到输出序列 (如翻译结果, 摘要) 的模型. 最早用于机器翻译, 给定一个源语言的句子, 自动翻译成目标语言.

模型结构

一个示例:

  • <bos> 和 <eos> 是自然语言处理中两个重要的特殊标记:
    • <bos>, Begin Of Sentence, 表示一个序列的开始
    • <eos>, End Of Sentence, 表示一个序列的结束

对这个结构的大致理解是, 用 RNN 把 input sequence 的信息压缩到 hidden state 中, 然后用另一个 RNN 从 hidden state 中提取信息输出.

可以接受任意长度的输入, 产生任意长度的输出.

构建 Seq2Seq 模型时, 更常用下面的结构:

  • Encoder 的输出会与 Decoder Embedding 输出拼接后输入 Decoder 的 RNN

对比 Autoencoder

Autoencoder 和 Seq2Seq 模型两者都有 Encoder-Decoder 结构:

特性 Autoencoder Seq2Seq
主要目标 数据压缩、降维、特征提取 序列转换（如翻译、摘要）
输入输出关系 输入 ≈ 输出（重建） 输入 ≠ 输出（映射到新序列）
典型结构 对称的编码器-解码器 非对称的编码器-解码器（常含注意力）
数据形式 非序列（图像/向量）或序列 必须为序列（文本、时间序列等）
训练目标 最小化重建损失（如MSE） 最小化序列预测损失（如交叉熵）
