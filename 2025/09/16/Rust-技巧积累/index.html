常用数学函数和常量

Rust 中常用的数学函数和常量定义在 std::f32 和 std::f64 中, 比如 :

  • $\pi$, std::f64::consts::PI
  • $e \approc 2.71828$, std::f64::consts::E

在浮点数 f32 和 f64 上有一些方法可用于计算:

  • .sqrt(), 平方根
  • .sin(), .cos(), .tan(), 三角函数
  • .exp(), $e$ 的 x 次幂
  • .ln(), 以 $e$ 为底的自然对数
  • .log10(), 以 $10$ 为底的自然对数
  • .powf(), 计算次方, 整型可以用 .pow()

获取 Iterator 的方法

Rust 标准库实现的 collections (如 Vec<T>, &[T], HashMap 等) 都能转换为 iterator, 用于遍历.

常见的获取迭代器的方法有 3 种:

  • iter(), 调用对象是 &T, 返回的迭代器类型 Iter<'_, T>, 此时元素类型都是 &T (引用), 无法修改
  • iter_mut(), 调用对象是 &mut T, 返回的迭代器类型 IterMut<'_, T>, 此时元素类型都是 &mut T (引用), 可以修改元素
  • into_iter(), 调用对象是 T, 返回的迭代器类型 IntoIter<T>, 此时元素类型都是 T, 会获取元素所有权
