获取 Iterator 的方法

Rust 标准库实现的 collections (如 Vec<T>, &[T], HashMap 等) 都能转换为 iterator, 用于遍历.

常见的获取迭代器的方法有 3 种:

  • iter(), 调用对象是 &T, 返回的迭代器类型 Iter<'_, T>, 此时元素类型都是 &T (引用), 无法修改
  • iter_mut(), 调用对象是 &mut T, 返回的迭代器类型 IterMut<'_, T>, 此时元素类型都是 &mut T (引用), 可以修改元素
  • into_iter(), 调用对象是 T, 返回的迭代器类型 IntoIter<T>, 此时元素类型都是 T, 会获取元素所有权
