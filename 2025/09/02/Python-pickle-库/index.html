Python-pickle-库

介绍

pickle 是 Python 序列化和反序列化标准库, 用于保存模型, 数据集或其他复杂对象.

Pickle 支持所有 Python 对象, 存储为二进制, 空间占用小, 但由于可执行代码, 不够安全.

  3. 触发 __reduce__ (自定义序列化行为) 等魔术方法
