RustSBI-基础

RustSBI 官方 startup

介绍

RustSBI 是 RISC-V SBI (Supervisor Binary Interface) 的第三方 Rust 实现.

SBI calls 和系统调用类似, 用 SBI extension 定义参数列表以及 function id 然后调用.

调用 SBI calls 需要先用 ecall 指令陷入到 M mode (Machine mode).

参数传递

extention id 会存放在 a7 寄存器, function id 存放在 a6, 参数列表存放在 a0-a5.

返回值

所有 SBI calls 都会返回两个 integers, 分别存储在 a0 和 a1 寄存器中:

  • Error number
  • Return value
