介绍

core 库是 Rust 中最底层的模块, 是标准库的一个子集, 能够在 #![no_std] 环境下运行, 提供了不依赖操作系统或标准库的类型, 函数和 trait. 其没有链接到上游库, 也没有 libc.

std 依赖 core, 但 core 不依赖 std.
