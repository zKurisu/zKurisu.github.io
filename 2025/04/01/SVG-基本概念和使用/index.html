SVG-基本概念和使用

介绍

SVG, Scalable Vector Graphics, 可缩放矢量图, 是一种特殊的描述二维图形的格式, 基于 XML.

主要特点是:

  • 矢量图形, 用数学公式描述图形, 而非像素点, 可以做到无限缩放而不失真
  • 基于 XML, SVG 文件是纯文本文件, 用 XML 语法编写
  • 可搜索性, 由于其纯文本, 可以被搜索引擎搜索

基本结构

最小的结构为:

  • xmlns 属性定义 SVG 命名空间 (相当于唯一标识一个 SVG 图片)

可以在 svg 标签上定义画布信息:

  • width/height 定义 viewport 的尺寸, 是在 HTML 中实际占据的物理显示区域大小
  • viewBox, 指 SVG 内容的虚拟坐标系统, 相当于相机的镜头 (框与 viewport 重合), 相机前是 SVG 内容, 可以平移和缩放, 格式为 viewBox="min-x min-y width height":
    • min-x 和 min-y 控制 “镜头对准哪里”, 0 0 表示不平移
    • viewportWidth / viewBoxWidth 表示 width 的缩放
    • viewportHeight / viewBoxHeight 表示 height 的缩放

基本图形

矩形 <rect>

  • x, y, 左上角的坐标
  • width, height, 宽高
  • fill, 填充的颜色
  • stroke, 描边的颜色
  • stroke-width, 描边的宽度

圆形 <circle>

  • cx, cy (center), 圆心的坐标
  • r, 半径

椭圆 <ellipse>

  • rx, ry, x 轴半径, y 轴半径
  • opacity, 透明度

直线 <line>

  • x1, y1, 起点坐标
  • x2, y2, 终点坐标

折线 <polyline>

  • points 中定义一系列点的坐标

多边形 <polygon>

  • 第一个点和最后一个点会闭合

路径 <path>

<path> 可以用来绘制任意形状的路径 (如直线, 曲线, 多边形等所有基本图形)

  • d, data, 由一系列绘图指令和坐标组合构成
    • M, move, 移动到 (Move to), M10 10, 移动到 (10, 10)
    • L, line, 画线到 (Line to), L100 100, 画直线到 (100, 100)
    • H, horizontal, 水平线
    • V, vertical, 垂直线
    • C: 三次贝塞尔曲线
    • S: 平滑三次贝塞尔曲线
    • Q: 二次贝塞尔曲线
    • T: 平滑二次贝塞尔曲线
    • A: 椭圆弧
    • Z: 闭合路径

文本 <text>

  • font-family 指定字体
  • font-size 指定字号
