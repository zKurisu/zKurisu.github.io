TeX 和 LaTeX 的关系

TeX 是最底层的引擎 (Donald Knuth 开发), LaTeX 是高级封装 (Leslie Lamport 开发). 类似于 “汇编语言” 和 C 的关系.

命令

命令, Command, 以 \ 开头, 用于执行特定操作. 一般分三类:

  • 无参数, 如 \LaTeX, 打印 LaTeX
  • 需必选参数, 如 \textbf{word}, 加上粗体
  • 可选参数, 如 \section[短标题]{长标题}

可以用 \newcommand 自定义命令:
