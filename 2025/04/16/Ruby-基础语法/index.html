类和对象

用 class 关键字定义:

  • @@ 开头的变量是 “类变量”, 所有实例共享
  • @ 开头的变量是 “实例变量”, 存在于单个实例中
  • new 是预定义的方法, 用于创建实例. 当传递有参数时, 则背后调用自定义的 initialize 方法

变量

Ruby 支持 5 种变量:

  • 小写字母, 下划线开头: 变量 (variable)

  • $ 开头: 全局变量 (global variable)

  • @ 开头: 实例变量 (instance variable)

  • @@ 开头: 类变量 (class variable)

  • 大写字母开头: 常量 (constant)

特殊变量

  • self: 当前方法的接收器对象
  • true: 代表 true 的值
  • false: 代表 false 的值
  • nil: 代表 undefined 的值
  • __FILE__: 当前源文件的名称
  • __LINE__: 当前行在源文件中的编号
