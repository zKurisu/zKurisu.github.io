  • msg db "hello, world!", 等价于 msg = "hello, world!"
    • msg 是变量名
    • db (define byte, 8 位) 是数据大小定义, 告诉汇编器后续数据是字节序列, 便于开辟空间, 除此之外还有 dw (define word, 16 位), dd (define double, 32 位), dq (define qxxx, 64 位)

rax, rdi, rsi, rdx 是用于系统调用的寄存器, 作用分别为:

  • rax, 临时寄存器, 当 syscall 时, 其必须包含 syscall number
  • rdi, 存储传递给函数的第一个参数
  • rsi, 存储传递给函数的第二个参数
  • rdx, 存储传递给函数的第三个参数

因此, 这部分等同于一次 sys_write 系统调用:

  • fd, 文件描述符, 0 表示标准输入, 1 代表标准输出, 2 代表标准错误
  • buf 字符数组指针, buf 存储的内容来自被指向的文件, fd 获得这些存储的内容
  • count, 由字符数组写入文件的字节数

堆栈

堆栈主要用于: 调用一个函数时, 把当前 RIP 内的地址入栈. 当函数执行结束时, 出栈栈顶地址到 RIP 寄存器, 以便程序在函数调用后继续执行, 如:

堆栈指针 stack pointer

  • rsp, stack pointer, 指向栈顶
  • rbp, base pointer, 指向栈底

与之相关的指令有:

  • push argument, 会将 rsp 自增 1, 同时存储 argument 到 rsp 指定的地方
  • pop argument, 从 rsp 指向的地方复制数据到 argument
