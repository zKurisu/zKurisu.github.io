介绍

FRR 是一个开源的高性能的 IP routing suite, 实现了所有标准的 routing protocols 如 BGP, RIP, OSPF, IS-IS, 且支持大量硬件设备.

FRR 在网络中负责与其他 routers 交换路由信息并向其他 layers 下发决策. 最常见的场景是将决策下发到 OS kernel.

Architecture

传统的 routing software 是用单一的 process 处理所有 routing protocol. FRR 是一组 daemons 协作来构建 routing table, 每一个协议由各自的 daemon 处理, 然后这些 daemon 与 middleman daemon (zebra) 通信.

一个 protocol daemon 出错不会影响其他 daemon 的运行, 且便于扩展新的协议.

FRR 用一个集成的 user interface 来与 daemons 交互 – vtysh. 其会用 UNIX socket 连接其他 daemon 作为代理进行配置.

vtysh 提供单一的配置文件来配置所有的 daemons, 避免分别维护各 daemon 的开销.
