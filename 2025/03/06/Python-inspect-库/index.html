Python-inspect-库

介绍

inspect 库是 Python 标准库中的模块, 用于 “获取对象的运行时信息” (模块, 类, 函数, 方法等), 可以检查对象的属性, 源代码, 参数信息等.

基本使用

检查对象类型

  • inspect.ismodule(obj) 检查是否为模块
  • inspect.isclass(obj) 检查是否为类
  • inspect.isfunction(obj) 检查是否为函数
  • inspect.ismethod(obj) 检查是否为方法
  • inspect.isbuiltin(obj) 检查是否为内置函数或方法

获取源代码

  • inspect.getsource(obj) 获取源代码字符串, 保留多行格式

  • inspect.getsourcelines(obj) 返回一个元组, 第一个元素是按行获取源码构成一个字符串数组, 第二个元素是源码在文件中的起始行号

获取参数信息

  • inspect.signature(obj), 获取函数或方法的参数签名

获取调用栈信息

  • inspect.curentframe() 获取当前帧对象
  • inspect.getouterframes(frame) 获取当前帧的外层调用栈信息
  • inspect.stack(), 获取当前调用栈信息

获取类或对象的成员信息

  • inspect.getmembers(obj), 获取对象的所有成员 (包括属性和方法)
  • inspect.getmro(cls), 获取类的继承顺序

Python 的反射机制

反射 (Reflection) 是一种动态获取和操作对象 (如类, 实例, 模块等) 的属性和方法的机制. 反射允许程序在运行时检查对象的类型, 属性, 方法等信息, 并动态地调用或修改它们.
