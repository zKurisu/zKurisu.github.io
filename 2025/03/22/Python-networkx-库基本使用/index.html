Python-networkx-库基本使用

networkx 官方网站
官方 Tutorial

介绍

networkx 是一个用于创建, 操作和研究复杂网路的 Python 库, 其支持多种类型的图, 如:

  • 无向图, 有向图, 多重图等

安装

基本概念

Multigraph

Multigraph, 多重图, 指一个图中可以包含多条连接同一对顶点的边.

Simple Graph

Simple Graph, 简单图, 每对顶点之间最多只有一条边.

Hashable object

在 Python 中, Hashable object 指能够被哈希函数处理的对象.

哈希函数会将对象映射为一个固定长度的整数值 (称为哈希值), 这个值通常用于快速查找和比较对象.

一般需要:

  • 实现了 __hash__() 方法, 其返回对象的哈希值
  • 实现了 __eq__() 方法, 用于比较两个对象是否相等

这些对象必须是 immutable, 即对象的内容在创建后不能被修改. (毕竟改了之后 hash 值也会改变)

常见的 hashable object 有:

  • int, 如 hash(42)
  • string, 如 hash("hello")
  • tuple, 如果 tuple 中所有元素都是 hashable, 则这个元组是 hashable, 如 hash((1, 2, 3))
  • float, 如 hash(3.14)
  • bool, 如 hash(True)
  • frozenset, 如 hash(frozenset({1,2,3}))

常见的 non-hashable object 有:

  • list, 可变的
  • dict, 可变的

Iterable container

在 Python 中, Iterable Container 指实现了 __iter__() 或者 __getitem__() 方法的对象, 其返回一个 iterator, 用于遍历.

示例:

  • __iter__() 定义 for xx in xx 等迭代时使用的迭代器
  • __contains__() 定义的是利用 in 判断元素是否存在的行为

Path Graph

Path Graph, 路径图, 指一种简单的线性图, 由一系列依次连接的节点组成, 每个节点 (除了首尾节点) 只与两个相邻的节点相连.
