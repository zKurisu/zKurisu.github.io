相关概念

Tween

Tween, 补间动画, 是 GSAP 中最基本的动画单元, 其定义了从一个状态到另一个状态的过渡过程. 如:

  • 将一个元素从 x: 0 移动到 x: 500
  • 将一个元素的透明度从 1 变为 0
  • 将一个元素的背景颜色从红色变为蓝色

本质是在指定时间内, 将一个对象的属性从起始值变化到目标值.

主要用:

  • gsap.to(selector, configOBJ)
  • gsap.from(selector, configOBJ)
  • gsap.fromTo(selector, configOBJInit, configOBJEnd)
    来创建 Tween.

configOBJ 中处包含一些 CSS 属性 (不一定相同, 比如 x 代表 translateX) 以外, 一些核心属性有:

  • duration, 动画的持续时间 (s)
  • delay, 动画开始前的延迟时间 (s)
  • ease, 控制动画变化的速度, 可以设置为 power2, bounce 等
  • repeat, 动画重复的次数 (2 表示 2 次, -1 代表无限次)
  • yoyo, 是否来回播放动画 (true 表示往返播放)
  • onStart, 动画开始时的回调函数
  • onComplete, 动画完成时的回调函数

Timelines

Timelines 是一个存放多个 Tween 的容器, 其将多个 Tween 组织在一起, 精确控制它们的播放顺序, 时间关系和整体流程,

或者直接创建并添加 Tween, 其与创建 Tween 所达到的效果一样:

  • tl.to(selector, configOBJ)

  • tl.from(selector, configOBJ)

不同的是, 可以指定 Tween 在 Timeline 中的开始时间, 如:

  • 指定相对时间, +=1 (上一个 Tween 结束后 1 秒) 或 -=1 (上一个 Tween 结束前 1 s)
  • 并行播放, < (与上一个 Tween 同时开始), <+0.5 (在上一个 Tween 开始 0.5s 后开始)
