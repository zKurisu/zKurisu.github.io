斐波那契数列是递归方式定义的整数数列, 前两项为 0 和 1, 后续每一项是前两项之和.

数学表达:
$$
\displaylines
{
\begin{aligned}
F_0 = 0, F_1 = 1 \newline~ \newline
F_n = F_{n-1} + F_{n-2}\ (n \ge 2)
\end{aligned}
}
$$

输入是数列的长度, 然后求完整的数列.

阶乘定义为: 正整数 n 的阶乘是所有小于等于 n 的正整数的乘积, 记作 n!.

数学表达;
$$
\displaylines
{
\begin{aligned}
0! = 1\ (特殊定义) \newline~ \newline
n! = n \times (n-1)!\ (递归定义) \newline~ \newline
n! = \prod_{k = 1}^n k\ (迭代定义)
\end{aligned}
}
$$

素数判断

素数判断指: 一个数 n > 1 是否只能被 1 和自身整除.

有一个定理: 威尔逊定理, p 是素数, 则有 $(p-1)! = -1\ mod\ p$

最大公约数

最大公约数指: 能同时整除多个整数的最大正整数.

计算方法通常是欧几里得算法, gcd(a,b) = gcd(b, a mod b).

输入两个整数, 找到最大公约数

最小公倍数

Lease Common Multiple, LCM, 最小公倍数, 指两个或多个整数共有的最小的正整数倍数, 简单来讲, 就是它是能同时整除这些数的最小的数.

比如: 求 4 和 6 的最小公倍数:

  • 4 的倍数: 4, 8, 12, 16, 20, 24 …
  • 6 的倍数: 6, 12, 18, 24, 30 …
  • 12 是第一个同时出现在两个列表中的数, 因此 LCM(4,6) = 12

可以利用最大公约数 (GCD) 来求解:

  1. 先求出两数的最大公约数 (GCD)
  2. 用两数的乘积处以 GCD 得到 LCM

$$
\displaylines
{
\begin{aligned}
LCM(a,b) = \frac{a \times b}{GCD(a,b)}
\end{aligned}
}
$$

比如求 15 和 20 的 LCM:

  1. $GCD(15, 20) = 5$
  2. $LCM = \frac{15 \times 20}{5} = 60$

标准的解决思路是用 stack, 时间和空间复杂度通常是 O(n). 核心思想:

  1. 遇到左括号: 如 (, [, {, 将其压入栈
