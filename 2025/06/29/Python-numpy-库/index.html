介绍

NymPy (Numerical Python) 提供高维数组 (ndarray) 和数学函数用于科学计算.

NumPy 只有一部分是 Python 编写, 设计高速计算的部分用 C/C++ 编写.

关键概念

Array

NumPy 提供的 array 对象 (ndarray) 比原生的 python list 快 50 倍以上.

ndarray 存储的元素类型必须相同, 且用连续的内存块存储数据.

Python list 允许类型可以不同, 但分散存储, 每个元素是独立对象, 因此占用更大.

Data types

Python 中的数据类型有:

  • strings, 如 "ABCD"
  • integer, 如 1, -1, 10
  • float, 如 1.2, 40.25
  • boolean, 如 True, False
  • complex, 如 1.0 + 2.0j

NumPy 提供而额外的数据类型, 用一个字符表示, 如 i 表示 integer, u 表示 unsigned integer.

  • i - integer
  • b - boolean
  • u - unsigned integer
  • f - float
  • c - complex float
  • m - timedelta
  • M - datetime
  • O - object
  • S - string
  • U - unicode string
  • V - fixed chunk of memory for other type ( void )
