Love2d-particle-system

介绍

Particle System 通过大量微小粒子的集体行为, 来模拟动态视觉效果, 如火焰, 烟雾, 魔法特效等.

核心术语有:

  • Particle, 粒子, 系统的基本单元, 通常为带属性的 2D 四边形 (含位置, 速度, 颜色, 生命周期等)
  • Emitter, 发射器, 控制粒子的生成规则 (如数量, 方向, 初始属性)
  • Controller, 控制器, 动态调整粒子的行为 (如重力影响, 颜色渐变等)

创建粒子系统

Love2D 中提供 love.graphics.newParticleSystem 接口来创建一个粒子系统. 即一个 ParticleSystem 类的对象.

  • texture 指 particle 的素材
  • buffer 指同一时间最多生成的 particle 数量

设置 particle 存在时间

  • min 指最短时间
  • max 指最长时间

设置每秒发射 particle 的数量

  • rate 为 particles per second

设置 particle 的加速度

  • xmin 是 x 轴的最小加速度
  • ymin 是 y 轴的最小加速度
  • xmax 是 x 轴的最大加速度
  • ymax 是 y 轴的最大加速度

设置 particle 的颜色

设置颜色:

  • r1, g1 这些的取值都为 0~1
  • 每 4 个为 1 组, 这些颜色会在 particle 的生命周期内遍历

设置 particle 生成的范围

  • distribution 有几个取值:
    • uniform
    • normal
    • ellipse
    • borderellipse
    • borderrectangle
  • dx 是生成 particle 的位置离 emitter 的最大 x 轴距离 (发射区域的 x 轴半径)
  • dy 是生成 particle 的位置离 emitter 的最大 y 轴距离 (发射区域的 y 轴半径)
  • angle 指发射区域的角度
  • directionRelativeToCenter, 若为 false, 粒子方向与 angle 定义的基准方向一致, 设置为 true 粒子方向则相对于发射区域的中心位置计算
