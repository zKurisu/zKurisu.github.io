介绍

Matplotlib 是一个常用的绘图库, matplotlib.pyplot 是其中最常用的模块, 提供了类似 MATLAB 的绘图接口.

重要类

Figure

Figure 类是 matplotlib 中最顶层的容器, 相当于一张 “画布”.

一个 Figure 可以包含多个 Axes (坐标轴, 子图), 但至少包含一个 Axes.

Figure 可以控制全局属性, 如图表大小, 背景色, 标题等.

Axes

Axes 类, 是 Figure 中的子图, 即实际绘制数据的区域.

每个 Axes 对象代表一个独立的图表.

Axis

Axis 类用于控制坐标轴, 如 scale, limits, generate ticks, ticklabels 等.

Artist

Artist 类, 指 Figure 中所有可见的部分, 包括 Figure, Axes, Axis, Text, Line2D 等等.

Artist 一般与单个 Axes 绑定, 不能共享和转移.

可用颜色

大部分绘图方法如 plot, scatter 都有 c (“color”) 参数, 可指定的颜色可以是:

  1. 基础颜色名称, 如:

  2. 缩写字符

  3. 十六进制颜色码, #RRGGBB 或者 #RRGGBBAA 格式

  1. (R, G, B) 或 (R, G, B, A) 元组, 范围都是 0~1

还有其他的没列出.

绘制散点图

绘制散点图其实就是把 plot 方法替换成 scatter 方法:

  • s 设置点的大小
  • alpha 设置透明度
  • 可以用 color 添加颜色
  • 用 marker 指定样式, o 表示圆形, s 表示方形
