构建 lag 数据

用 DataFrame 或 Series 的 shift 方法, 其沿索引方向移动 n 个位置:

  • n > 0, lag, 数据向下移动
  • n < 0, lead, 数据向上移动
  • n = 0, 数据不变

数据对齐

  • y 虽然是 Series, 但也有 index 列
  • align() 用于确保两个 DataFrame/Series 的索引一致, 去除不一致的索引并确保行数一致
  • join 有几个参数:
    • inner 表示交集, 只保留两者共有的索引
    • outer 保留所有索引, 缺失值用 NaN 填充
    • left 以左边的索引为准
    • right 以右边的索引为准
