简介

Scikit-learn, 简称 sklearn, 是一个流行的机器学习库, 其提供丰富的工具用于数据挖掘和分析, 其基于 NumPy, SciPy 和 Matplotlib 构建.

拟合和预处理

SKlearn 实现了大量机器学习的算法和模型, 称 “estimators”, 每个 estimator 能够通过 fit 方法来利用数据进行拟合.

  • X 指数据集, 一般是二维数据, 第一层是样本, 第二层是 features
  • y 指目标集 (target value), 对于 regression 任务而言是一组 real numbers, 对于 classification 任务而言是一组离散值, 如果是 unsupervised learning tasks 则不需要指定

Transformers 和 pre-processors

Machine learning 任务通常分多个阶段, 一般需要 pre-processing 来处理或转换数据.

Sklearn 中的 pre-processors 和 transformers 与 estimator 对象的接口大致相同 (都继承自 BaseEstimator 类), 但 predict 方法换成了 transform 方法来产生新的样本数据.
