Deep-Learning-入门

介绍

Deep Learning 是 Machine Learning 的一个子领域, 用多层神经网络自动学习数据特征表示. 不需要人工设计特征 (不同于 ML 需要特征工程), 模型直接从原始数据中学习抽象特征.

Deep Learning 需要海量数据才能发挥优势, 且模型极其复杂, 需要高性能 GPU/TPU, 内部机制难以解释, 但能够处理非结构化数据并进行复杂模式识别.

Deep Learning 一般也分为:

  • 监督学习, 如 CNN (图像), RNN/LSTM (时序), Transformer (NLP)
  • 无监督学习, 如 Autoencoder, GAN (生成对抗网络)
  • 强化学习, 如 AlphaGo

相关概念

Neuron

每一个 Neuron 代表一个简单运算.

一个包含一个输入的 neuron 如下图:

  • x 是输入

  • b 是 bias, 用于调整输出
  • y 是最终输出, 这里 $y = w * x + 1 * b$

一个 neural network 学习的过程就是调整 weight.

Learning curves

Learning curves 描述 training 和 validation 的 loss 值随 epochs (或训练数据量) 的变化情况:

  • signal 指训练集中能让模型预测新数据的信息
  • noise 指训练集中只能让模型预测训练集内容的信息
  • 只有当 model 从 signal 中学习时, validation 的 loss 曲线才会下降
  • 遇到 noise 时会形成 gap

Capacity

Capacity 指模型能够学习的 partterns 的复杂程度以及大小, 其很大程度上取决于 neurons 的数量以及连接方式.

当 underfitting 时, 就需要增加 capacity: 添加更多 layers (deeper) 或者每个 layer 添加更多 units (wider).

神经网络的处理方法

神经网络是按行输入样本, 比如 shape 为 (1, features), 还是说按 batch 输入样本, 比如 shape 为 (batch_size, features)?

单样本输入 (逐行处理) 一般只会出现在在线学习或实时推理场景, 缺点是无法利用硬件并行计算, 效率极低:

批量处理 (batch processing), 默认一次处理多个样本, 比如 shape 为 (batch_size, features), 优点是能够并行运算, 比如 GPU 同时处理整个 batch, 显著提升速度, 且 batch 内样本的梯度平均后更新参数能减少噪声:

框架底层实现

如 PyTorch/TensorFlow 等框架, 会将 batch 内所有样本的运算向量化, 利用矩阵乘法一次性处理整个 batch, 比如:

$$
\displaylines
{
\begin{aligned}
Z = XW + b, X \in R^{batch_size \times features}, W \in R^{features \times hidden_dim}
\end{aligned}
}
$$
