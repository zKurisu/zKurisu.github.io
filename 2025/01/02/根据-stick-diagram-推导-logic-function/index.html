根据-stick-diagram-推导-logic-function

示例图:

  • A 和 B 的 Polysilicon 与竖直线的每一个交点, 就代表一个 MOS.
  • 一头与 VDD 相连的 MOS 为 PMOS
  • 一头与 GND 相连的 MOS 为 NMOS
  • 两个 MOS 位于两个竖直线为并联
  • 两个 MOS 位于同一竖直线为串联

推导过程:
要根据 stick diagram 推导逻辑函数，需要分析 PMOS 和 NMOS 网络的结构。这些网络分别定义了逻辑函数的 “拉高” 和 “拉低” 过程。我们逐步分析该图的 PMOS 和 NMOS 网络，并推导出逻辑功能。

1. PMOS 网络分析（从 VDD 到输出 X）

PMOS 网络通过高电平连接到输出 X，用于“拉高”操作。PMOS 的导通条件是输入端为低电平。

PMOS 网络结构：

  • 上图显示了 PMOS 网络中两个 PMOS 器件：
    • 一个 PMOS 的源极连接到 VDD，其栅极由输入 A 控制。
    • 另一个 PMOS 的源极也连接到 VDD，其栅极由输入 B 控制。
  • 两个 PMOS 器件是并联的。

并联 PMOS 的逻辑功能：

  • 并联 PMOS 表示：输出 X 被拉高到 VDD，当 A = 0 或 B = 0。
  • 逻辑表达式：
    $$
    \displaylines
    {
    \begin{aligned}
    X = A’ + B’
    \end{aligned}
    }
    $$

（注意：PMOS 的逻辑是输入信号的反相）

2. NMOS 网络分析（从 GND 到输出 X’）

NMOS 网络通过低电平连接到输出 **X’**，用于“拉低”操作。NMOS 的导通条件是输入端为高电平。

NMOS 网络结构：

  • NMOS 网络由两个 NMOS 器件组成：
    • 一个 NMOS 的源极连接到 GND，其栅极由输入 A 控制。
    • 另一个 NMOS 的源极也连接到 GND，其栅极由输入 B 控制。
  • 两个 NMOS 器件是串联的。

串联 NMOS 的逻辑功能：

  • 串联 NMOS 表示：输出 X’ 被拉低到 GND，当 A = 1 且 B = 1。
  • 逻辑表达式：
    $$
    \displaylines
    {
    \begin{aligned}
    X’ = A \cdot B
    \end{aligned}
    }
    $$

3. 综合 PMOS 和 NMOS 网络

PMOS 和 NMOS 网络是互补的，形成 CMOS 逻辑电路。

  • PMOS 网络的输出 $X = A’ + B’$
  • NMOS 网络的输出 $X’ = A \cdot B$
  • CMOS 电路遵循逻辑恒等式 (上面两个任一个加上非得到的, 比如对 NMOS 网络: $X=\overline{A \cdot B} = (A \cdot B)’$)
    $$
    \displaylines
    {
    \begin{aligned}
    X = (A \cdot B)’
    \end{aligned}
    }
    $$

4. 逻辑功能

最终的逻辑功能为：
$$
\displaylines
{
\begin{aligned}
X = (A \cdot B)’
\end{aligned}
}
$$
该逻辑功能表示 **NAND (与非门)**。
