相关概念

Node

Node 指网络中的 hosts, switches, controllers 等. Local nodes 指 local machine 上的一些进程.

mininet.node

用于创建和管理 Mininet 中的 node objects. 其提供几个有用的类:

  • Node, 所有 nodes 的父类
  • Switch, 所有 switch nodes 的父类
  • Controller, 所有 OpenFlow controllers 的父类
  • Host, 虚拟主机, 一般是一个 shell. 可以用 Cmd, sendCmd() 来发送指令来知执行

Mininet

addController 方法, 往网络中添加 controller

addSwitch 方法, 往网络中添加 switch

addHost 方法, 往网络中添加 host

addLink 方法, 往网络中添加链路

start 方法

其会自动完成下面操作:

  1. 调用 build() 方法构建拓扑
  2. 启动所有控制器 (调用 controller 的 start 方法)
  3. 启动所有交换机，并将它们连接到指定的控制器 (调用 switch 的 start 方法)
  4. 配置主机和交换机的网络接口
  5. 启动主机并分配 IP 地址
    (Host 不需要手动唤醒)

build 方法

用于构建网络拓扑, 但不会启动任何设备.

stop 方法

关闭所有运行的设备 (如控制器, 交换机, 主机等), 并清理虚拟网络环境.
