L03 Performance

Cycles per Instruction

Average Cycles per Instruction, 平均一个 instruction 所需的 cycle 数:
$$
\displaylines
{
\begin{aligned}
CPI = \frac{\sum_{i=1}^{n} (CPI_i \times I_i)}{I_C}
\end{aligned}
}
$$

  • $CPI_i \times I_i$ 指某种类型的 Instruction 所花的总 cycle 数
  • $I_C$ 指 Instruction 的总数

Processor time $T$ (处理一个 programm 的总时间):
$$
\displaylines
{
\begin{aligned}
T = I_C \times CPI \times \tau
\end{aligned}
}
$$

  • $I_C \times CPI$ 指总 cycles 数
  • $\tau$ 是一个 cycle 对应的时间
