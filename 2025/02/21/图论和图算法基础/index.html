1 图的基本概念和数据结构

  • v (vertice) 用于表示顶点
  • V 表示顶点的集合
  • e (edge) 表示边
  • E 表示边的集合
  • G 是 V 和 E 的二元组

Undirected 和 Directed

Undirected 指无向图, 两顶点间的边没有 “指向” 作用, 可以视作是双向的:

Directed Graph 指有向图, 每条边都指示方向, 为单向的:

Unweighted 和 Weighted

Unweighted 指, 每一条边的权重一样, 相当于没有权重:

Weighted 指, 每一条边有权重, 大小可以不同, 在不同场景下, 权重代表不同的物理意义:

Adjacency list

Adjacency list (邻接表), 记录每一个 vertex 相邻的有那些 vertices:

  • 1 指 $v_1$, 其相邻的有 2,3,4 即 $v_2,v_3,v_4$

Adjacency matrix

Adjacency matrix (邻接矩阵), 大小为 m x m (m 是顶点数), 每一个元素表示两顶点间边的权重:

  • 若权重为 0 表示没有边 (有时是无穷, 取决于具体的应用)
  • 对于无向图, 邻接矩阵都是对称的, 如 $e_{3,6}$, $e_{6,3}$ 相等
  • 对于无向图, 一般不对称, 因为如 $e_{3,6}$, $e_{6,3}$ 不一定相等

时间复杂度分析如下:

  • Dijkstra’s Algorithm 需要每一条边的权重为非负, 不然可能会永远循环下去
  • 总的 enqueue 和 dequeue 次数为 $O( \left\vert V \right\vert + \left\vert E \right\vert )$
  • 每次 enqueue 和 dequeue 操作的复杂度为 $O( \log \left\vert V \right\vert )$
  • 因此总的时间复杂度为 $O( (\left\vert V \right\vert + \left\vert E \right\vert ) \cdot \log \left\vert V \right\vert )$

  • Trees are connected acyclic undirected graphs

树中的节点和边有特殊的关系:

  • 如果有 $n$ 个 vertices, 则有 $n-1$ edges (少一条不连通, 多一条有回路)

Spanning Trees

从一个有 n 个 vertices 的 connected undirected graph 中找到满足:

  • 有 $n$ 个 vertices (即所有的节点都包括)
  • 有 $n-1$ edges (保证无回路)
  • 图中所有节点是连通的

的 subgraph 称为 spanning tree. (记住特点, 所有节点连通, 无回路) 从一个图中得到的 spanning tree 并不唯一.

6 Prim’s Algorithm

Prim’s Algorithm 用于寻找 minumum spanning tree.

其基本思想是, 从一个初始节点 (one vertex, no edge), 慢慢增长到一棵树 (spanning tree).

  • 初始, one vertex and no edge
  • 让 $U$ 包含 tree 的所有 vertices
  • 让 $T$ 包含 tree 的所有 edges
  • Iteration, 每次添加一个 vertex 和一个 edge 到 tree 中, 需要考虑如下条件:
    • 定义集合 $S = \{e_{xy} \in E | x \in U\ and\ y \notin U\}$, $x$ 节点在生成树节点集合内, $y$ 节点不在. $e_{xy}$ 表示两节点的边的权重
    • 选出集合 $S$ 中权重最小的边 $e_{uv}$
    • 将 edge $e_{uv}$ 加入集合 $T$
    • 将 vertex $v$ 加入集合 $U$
  • 算法循环的次数等于 vertex 的数量, 然后结束算法

注意每次循环需要保证树的两个性质:
* Connectivity
* No cycle

具体示例

Connected Dominating Set

Connected Dominating Set, 连通支配集, 是一个 graph 的 vertex set 的子集, 需要满足两个性质:

  1. Graph 中所有的 vertices 要么位于该 subset 中, 要么与该 subset 中的 vertices 邻接 (adjacent)
  2. 由该 subset 组成的 sub graph 是 connected 的

而 Minumum CDS (MCDS) 则是满足上述性质的最小 vertex 子集.

论文原文: A CDS is a subset of the vertex set of a graph which follows two properties. Firstly, each vertex of the graph either belongs to this subset or is adjacent to a vertex in the subset. Secondly, the sub graph induced by the edges between the nodes of this subset is connected. A Minimum CDS (MCDS) is a CDS with minimum number of nodes in it and it has been proved that computing a MCDS is a NP complete problem.

一个示例:

CDS based routing 就是指, 仅由 CDS nodes 来转发消息. 因此 CDS nodes 可以视作整个网络信息传输的 backbone. 这种路由方式可以减少网络中的 traffic 以及资源消耗.

另一个对 Dominating Set 的描述也挺好的:

A dominating set $D$ of $G$ is a subset of $V$ such that any node not in $D$ has at least one neighbor in $D$

Classification of CDS construction methods

CDS construction 可以分为两类:

  • Centralized, 需要预先知道整个网络的 topo, 才能得到 MCDS
  • Distributed, 每个节点需要基于本地信息决定自己是否属于 CDS, 又可以分为两类:
    • Pruning based CDS
    • Maximal Independent Set (MIS) based CDS

CDS construction 方面的工作最早由 Guha 和 Kuller 完成, 其引出两个 Greedy Heuristic algorithms (具体看论文 Approximation Algorithms for Connected Dominating Sets).

Unit Disk Graph

Unit Disk Graph 用于描述节点之间基于距离的通信关系:

  • 给定一个点集 $V$, 每个点代表一个网络节点
  • 每个节点视为一个半径为 1 的圆盘, 圆盘的中心是节点的位置
  • 如果两个圆盘相交 (即两个节点间的距离不超过 1), 则说明它们是相邻的, 在通信建模中可以视作俩节点能通信
  • 若不相邻, 则认为两个 nodes 是 independent

数学化的定义为:
$$
\displaylines
{
\begin{aligned}
G = (V,E), 其中\ E={(u,v) | distance(u,v) \le 1}
\end{aligned}
}
$$

Independent Set

Independent Set, 独立集, 指图中节点的一个子集, 其中任意两个节点之间没有边相连.

定义为:

  • 给定一个图 $G=(V,E)$, 其中 $V$ 是节点的集合, $E$ 是边的集合, 一个 Independent Set 就是 $V$ 的一个子集 $S$, 使得 $S$ 中任意两个节点之间没有边相连:
    $$
    \displaylines
    {
    \begin{aligned}
    \forall u,v \in S, (u,v) \notin E
    \end{aligned}
    }
    $$

Maximal Independent Set

Maximal Independent Set, MIS, 定义为:

  • 一个独立集 $S$ 是 MIS, 如果它不能再加入任何新的节点而不破坏其独立性

注意 Maximal Independent Set 不等价于 Maximum Independent Set

  • Maximal 可以是局部最大

原文: Two nodes are independent if they are not neighboring to each other. $S \subset V$ is an independent set of G if for $\forall u,v \in S$, $u$ and $v$ are independent. $S$ is maximal (MIS) if adding any node into $S$ breaks the independence property.

MIS 也是一个 dominating set.

在 unit-disk graph G 中, MIS 的大小与 MCDS 的关系为 [Distributed Construction of Connected Dominating Set in Wireless Ad Hoc Networks] 这篇文章还没下:

Let $S$ be any maximal independent set and $D$ be any MCDS of a unit-disk graph $G$. Then $\left\vert S \right\vert \le 4 \cdot \left\vert D \right\vert + 1 for \left\vert D \right\vert > 1$.

Steiner 树问题

Steiner 树问题, 指, 给定一组端点, 通过引入额外的中间点 (Steiner point), 使得连接所有端点的总权重 (如边的长度或 cost) 最小. 其属于 NP-Hard 问题.

如 4 个端点构成一个正方形, 用最小生成树 (MST) 来连接, 总长度是 3 条边之和 (假设边长为 1, 总长则为 3), 而 Steiner tree 中, 可以在中心添加一个 Steiner point, 通过两条对角线连接四个点, 总长度减少到 $2 \sqrt{2} \approx 2.828$:

Complete Graph

Complete graph 是一个无向图, 其中每对不同的顶点之间都有一条边相连, 如果一个图 $K_n$ 有 n 个顶点, 那么它有 $\frac{n(n-1)}{2}$ 条边.
